## 3. 游戏流程设计

> Game Flow Design

### 3.1 游戏总体粗流程

1. 入场动画 / 加载动画
2. 开始界面选择 **开始游戏** 或 **加载游戏**
3. 选择性别
4. 输入玩家名字
5. 选择录取专业
   - 生物专业，决定主线事件"选课"的名称（对应代码逻辑）
   - 目前开发 Demo 版本仅单一专业，无影响
6. **楔子 / 引子：** 经过了高考的洗礼终于考上了心心念念的大学，无论是第几志愿，最终与我们未来至少四年青春结缘的是南京农业大学。
7. **主线事件：** 你想要度过怎样的大学生活？（详见 [事件设计](#4-成就事件与结局)）
8. 进入游戏初始场景——宿舍（详见 [游戏地图设计](#9-游戏地图功能设计)）
9. 进入回合（详见 [游戏主线流程设计](#32-游戏主线流程设计)）
10. 每学年触发学年总结，评分学年
11. 结局判定：
    - 若有特殊结局，则优先触发特殊结局
    - 若无特殊结局，则触发正常结局 = 恋爱结局 + 毕业结局（详见 [成就、事件与结局](#4-成就事件与结局)）

### 3.2 游戏主线流程设计

#### 大一上

| 回合   | 时间 | 主线事件                                                                                                                                                                                                                                                 |
| ------ | ---- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 回合一 | 9月  | 大学报到、学费扣除（5000元）、选课、**神秘二手书学长**（概率触发，选课后、学杂费结算前）、学杂费结算（按科目书单扣费，购买了二手书的科目不再收费）、推销、入学典礼、**军训**（压缩为单一事件）、社团招新（强制）、你想要度过怎样的大学生活？ |
| 回合二 | 10月 | 班会、国庆假期、班委竞选（可选）                                                                                                                                                                                                                         |
| 回合三 | 11月 | 神秘二手书学长（概率触发）、**父母来电·人生规划**（影响结局）                                                                                                                                                                                     |
| 回合四 | 12月 | 四级报名、期末周突击复习、元旦/圣诞活动（可选）                                                                                                                                                                                                          |
| 回合五 | 1月  | 期末考试结算、校园跑结算、学期总结·GPA公示、寒假规划                                                                                                                                                                                                    |
