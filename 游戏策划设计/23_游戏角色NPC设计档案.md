# 人物设计文档

## 一、人物设计总述

本游戏以大学校园生活为核心叙事背景，围绕玩家进入校园后的学习、社团、生活、探索与人际关系展开。六位主要 NPC 分别对应校园生活中的不同功能场景与情绪体验：学习指导、运动活动、技术实践、传统文化、学生事务、校园探索。每位角色既承担剧情推进功能，也承担玩家情感连接与校园氛围塑造功能。

六位 NPC 的设计目标是形成清晰的角色差异，使玩家在游玩过程中能够通过角色外形、性格、语言风格、行为习惯和剧情任务快速识别不同人物，同时让他们共同构成一个完整、真实、具有青春感的校园关系网络。

角色整体采用统一的日系视觉小说 / 二次元手游校园剧情立绘风格，人物比例控制在约 6 到 6.5 头身，线稿清晰，色彩为低到中等饱和度，使用轻赛璐璐阴影。角色立绘需适合 Unity 2D 游戏中的对话界面、任务触发界面、好感度系统、校园探索互动等场景使用。

---

## 二、角色设计原则

### 1. 统一画风原则

所有主要 NPC 采用同一套 2D 游戏角色立绘风格：清晰线稿、中等饱和度配色、轻赛璐璐阴影、干净轮廓、全身立绘。角色不采用写实风格，也不采用厚涂插画风格，避免和游戏整体校园视觉风格产生冲突。

人物立绘主要用于 Unity 2D 对话系统和角色互动界面，因此需要保证轮廓清楚、姿势稳定、服装结构清晰，便于后期进行透明 PNG 抠图、Sprite 导入、对话框遮罩、表情差分制作等。

### 2. 差异化原则

六位 NPC 不能只是服装不同，而应在气质、功能、动作、语言风格、剧情任务和好感互动方式上形成差异。玩家需要通过短时间接触就能理解每个角色的定位。

| 角色 | 核心定位 | 玩家感受 |
|---|---|---|
| 顾景和 | 温柔学长型 | 安心、可靠、被引导 |
| 陆向阳 | 阳光运动型 | 热闹、轻松、有活力 |
| 程砚舟 | 冷静技术型 | 理性、聪明、关键时刻可靠 |
| 林知晚 | 温柔治愈型 | 柔和、安静、被陪伴 |
| 沈清妍 | 文雅学姐型 | 端庄、优秀、有距离感但可靠 |
| 陈书仟 | 元气行动型 | 主动、热情、校园探索感强 |

### 3. 校园功能对应原则

每个角色都对应一个或多个校园功能场景，使 NPC 不只是“对话角色”，还能够自然承担游戏任务系统的一部分。

| 角色 | 对应校园模块 |
|---|---|
| 顾景和 | 图书馆、学习指导、学生会、迎新引导 |
| 陆向阳 | 操场、篮球场、运动会、体育社团 |
| 程砚舟 | AI 工坊、机房、程序社团、技术任务 |
| 林知晚 | 传统文化社团、汉服活动、春日校园活动 |
| 沈清妍 | 学生会、文学社、活动策划、主线事务推进 |
| 陈书仟 | 校园探索、迎新活动、跑腿任务、社团活动 |

### 4. 好感与互动原则

角色设计应服务于好感系统。每位角色都有不同的喜好、雷区和互动节奏。玩家不能用同一种对话方式刷所有角色好感，而需要根据角色性格选择不同互动策略。

例如，顾景和重视责任感，沈清妍重视分寸感，程砚舟重视有效沟通，林知晚重视真诚与耐心，陆向阳重视积极参与，陈书仟重视回应与陪伴行动。

# 三、主要人物设计

---

## 1. 顾景和

### 基础信息

| 项目 | 内容 |
|---|---|
| 姓名 | 顾景和 |
| 性别 | 男 |
| 年级 | 大三 |
| 身份 | 学生会副部长 / 图书馆志愿者 / 新生学习指导员 |
| 角色类型 | 温柔学长型 |
| 主要活动区域 | 图书馆、教学楼、学生会办公室、自习室 |
| 核心关键词 | 温柔、可靠、耐心、稳重、礼貌、有责任感 |

### 角色定位

顾景和是玩家进入校园后较早接触到的学长型 NPC。他负责帮助玩家熟悉校园规则、学习安排、图书馆系统、学生会事务和基础任务流程。作为温柔可靠型角色，他在游戏前期可以承担引导玩家理解玩法机制的功能。

他不是强势的领导者，而是温和的协助者。他会提醒玩家该做什么，但不会以命令的方式推动玩家，而是通过耐心说明和合理建议让玩家自然接受任务。

### 人物介绍

顾景和给人的第一印象是干净、礼貌、可靠。他说话语速不快，语气温和，总是会先听完别人的问题再作出回应。在新生刚进入校园、对环境不熟悉时，他往往是最容易让人放下紧张感的人。

他习惯把事情提前安排好，也习惯承担别人没有注意到的细节。无论是活动签到表、图书馆借阅说明，还是迎新流程中的临时问题，他都能用稳定的方式处理。

虽然他看起来一直从容，但他并不是没有压力。他只是习惯把责任放在情绪前面，不轻易表现自己的疲惫。随着剧情推进，玩家可以逐渐发现他温柔背后隐藏的自我要求，以及他偶尔也需要被别人理解的一面。

### 外观设计

顾景和的立绘以浅米色针织开衫、白衬衫、深色长裤为核心视觉元素，整体颜色干净柔和，体现文雅、可靠的校园学长感。手持书本或文件夹，强化他学习指导员、学生会成员或图书馆常驻学长的角色身份。

发型为黑色或深棕色短发，干净自然，刘海整齐但不死板。面部表情以温和微笑为主，眼神亲切、有耐心，不应表现出过度成熟或教师感。

### 性格设计

顾景和温柔但不软弱，礼貌但有边界感。他愿意帮助别人，但不喜欢别人把他的帮助视为理所当然。他做事稳妥，对时间、承诺和责任比较看重。

他的温柔不是讨好型，而是一种成熟的分寸感。他知道什么时候该提醒别人，什么时候该保持沉默，也知道如何在不伤害对方自尊的情况下提出建议。

### 喜好

顾景和喜欢安静、有秩序的环境，例如图书馆、自习室、教学楼走廊和校园长椅。他喜欢阅读、整理资料、做计划、帮助别人解决实际问题，也喜欢温热饮品和整洁的笔记。

他欣赏认真、守时、有礼貌的人。如果玩家在任务中认真完成承诺，主动承担责任，或者在对话中表现出倾听和尊重，他会提升好感。

### 雷区

顾景和不喜欢随意爽约、不守时、临时甩锅的人。他也不喜欢别人一边请求帮助，一边完全不愿意自己努力。

他的雷区不是被麻烦，而是“被理所当然地依赖”。如果玩家经常让他收拾烂摊子，却不表达感谢或改正态度，他会逐渐变得疏远。

### 语言风格

顾景和说话温和、清楚、有条理。他不会使用夸张语气，也不会过度调侃玩家。他常用提醒式、建议式表达。

#### 示例台词

> “不用着急，先把要处理的事情按顺序列出来，会清楚很多。”  
> “如果你不确定怎么做，可以先从最简单的一步开始。”  
> “我可以帮你看一遍，但下次记得提前准备。”  
> “答应别人的事情，最好不要轻易拖到最后一刻。”

### 剧情功能

顾景和适合承担游戏前期引导功能，包括图书馆使用说明、课程安排任务、学生会基础任务和迎新活动支线。他也可以作为玩家遇到困难时的稳定支持角色。

| 剧情类型 | 示例 |
|---|---|
| 新手引导 | 带玩家熟悉图书馆、自习室、任务栏 |
| 学习任务 | 帮玩家整理复习资料、寻找课程信息 |
| 学生会任务 | 协助完成活动表格、签到安排 |
| 情感支线 | 发现他独自承担过多事务，玩家主动帮忙 |

### 好感互动方向

顾景和的好感线适合从“信任”开始，而不是从轻松玩笑开始。玩家认真完成任务、尊重他的时间、主动分担压力，会比单纯表达依赖更有效。

他的好感推进节奏应较为平稳，从普通学长到可靠朋友，再到愿意向玩家展现疲惫与脆弱的一面。

## 2. 陆向阳

### 基础信息

| 项目 | 内容 |
|---|---|
| 姓名 | 陆向阳 |
| 性别 | 男 |
| 年级 | 大二 |
| 身份 | 篮球社成员 / 体育部干事 / 运动会志愿者 |
| 角色类型 | 阳光运动型 |
| 主要活动区域 | 操场、篮球场、体育馆、社团活动区 |
| 核心关键词 | 阳光、开朗、直率、行动派、讲义气、有活力 |

### 角色定位

陆向阳是校园中的运动系活力角色。他适合负责运动会、操场探索、篮球社任务、校园活动挑战等剧情内容。与顾景和的稳重不同，陆向阳更强调行动力、热情和团队感。

他可以在游戏中承担节奏调节功能。当剧情过于安静或任务偏文书时，陆向阳的出现能够带来轻松、热闹和青春感。

### 人物介绍

陆向阳是那种很容易被记住的人。无论是在篮球场还是运动会报名点，他总是笑得很开朗，和谁都能很快聊起来。他不太喜欢把事情想得太复杂，更相信“先试试看”。

他有时候会显得有些冲动，但并不是不顾别人感受。他只是习惯用行动解决问题。遇到困难时，他通常不会先抱怨，而是直接问：“那我们现在能做什么？”

陆向阳很适合成为玩家在校园中的活动搭档。他可以带玩家参加操场任务、运动会支线、篮球社挑战，也可以用轻松的方式推动玩家认识更多校园地点。

### 外观设计

陆向阳的立绘以蓝白运动外套、深色运动裤、运动鞋和篮球为主要视觉元素。动作可以是挥手、单手拿篮球、插兜或叉腰。整体姿态应自然开朗，体现运动社团成员的亲和力。

他的发型为黑色或深棕色短发，略微蓬松，有运动感但不夸张。脸部表情以开朗微笑为主，可以微微露齿，但不能张嘴过大或显得幼稚。

### 性格设计

陆向阳直率、热情、行动力强。他不擅长复杂的情绪分析，但很擅长用陪伴和行动支持别人。他的安慰方式通常不是长篇道理，而是拉着对方去走一走、打一场球、完成一个小挑战。

他很讲义气，也很重视团队配合。虽然平时大大咧咧，但对朋友是否真心参与、是否认真对待约定很敏感。

### 喜好

陆向阳喜欢篮球、跑步、运动会、社团团建和热闹的校园活动。他也喜欢冰饮、运动饮料、食堂大份套餐、校园打卡和团队挑战。

他喜欢玩家积极回应他的邀请。如果玩家愿意陪他参加活动，或者在比赛、任务中认真投入，他会很快产生亲近感。

### 雷区

陆向阳不喜欢消极抱怨、临阵退缩和敷衍团队的人。他最不喜欢还没开始就说“不可能”。

他也不喜欢别人用“只会运动”“没脑子”“太吵”来评价他。虽然他表面很开朗，但如果被否定努力和热情，他也会受伤。

### 语言风格

陆向阳说话直接、明快，语气轻松。他经常用鼓励式表达，喜欢带动气氛。

#### 示例台词

> “别站着想啦，先试试看！”  
> “输了也没事，下次赢回来就行。”  
> “走，我带你去操场，那边今天可热闹了。”  
> “你这不是做得挺好吗？再自信一点啊。”

### 剧情功能

陆向阳适合负责运动、探索和团队活动相关剧情。

| 剧情类型 | 示例 |
|---|---|
| 校园探索 | 带玩家熟悉操场、篮球场、体育馆 |
| 运动会任务 | 报名、训练、比赛支线 |
| 社团活动 | 篮球社招新、社团挑战 |
| 轻松支线 | 陪玩家散步、买饮料、完成跑腿任务 |

### 好感互动方向

陆向阳的好感提升相对直接。玩家愿意参与活动、积极回应他的热情、在比赛中认真配合，就会明显增加好感。

他的好感剧情适合轻快、有青春热血感的节奏，也可以在后期加入“表面开朗但害怕拖累团队”的隐藏情绪，使角色更有层次。

## 3. 程砚舟

### 基础信息

| 项目 | 内容 |
|---|---|
| 姓名 | 程砚舟 |
| 性别 | 男 |
| 年级 | 大二或大三 |
| 身份 | AI 工坊成员 / 计算机社团核心成员 / 程序设计竞赛队成员 |
| 角色类型 | 冷静技术型 |
| 主要活动区域 | AI 工坊、机房、实验室、社团工作室 |
| 核心关键词 | 冷静、理性、聪明、专注、内敛、技术感、嘴硬心软 |

### 角色定位

程砚舟是技术型 NPC，适合承担游戏中的系统解谜、设备修复、代码任务、隐藏信息解锁、AI 工坊支线等内容。他代表校园中的技术实践与理性思考部分。

与顾景和的温柔可靠、陆向阳的热情行动不同，程砚舟的角色魅力来自冷静、聪明、专业和不轻易表露情绪。

### 人物介绍

程砚舟常出现在机房、实验室或社团工作室。相比热闹的活动现场，他更习惯待在安静、有设备和代码的地方。他不太擅长寒暄，也很少主动参与无目的的社交。

他说话直接，有时显得冷淡，但通常不是故意疏远别人，而是习惯用最短的话说清楚问题。只要涉及程序、设备、数据或系统故障，他总能很快抓住关键点。

熟悉之后会发现，他其实并不难相处。他属于“嘴上说麻烦，手上已经开始修”的类型。表面克制，内心有责任感，关键时刻非常可靠。

### 外观设计

程砚舟的立绘以深色轻便外套、白色或浅灰 T 恤、深色工装裤、细框眼镜、平板或笔记本电脑为主要视觉元素。整体偏深色系，但不能过度阴郁，需要保持校园清爽感。

动作可以是轻轻推眼镜、一手拿平板或笔记本电脑、一手插兜。姿态克制、安静，和陆向阳的挥手动作、顾景和的持书动作形成明显区别。

### 性格设计

程砚舟冷静、理性、专注。他不喜欢无效沟通，也不喜欢没有信息量的求助。他更愿意面对清晰的问题，而不是模糊的情绪表达。

他看似不好接近，但不是没有同理心。他只是表达方式更偏行动型。玩家遇到技术困难时，他可能嘴上嫌麻烦，却会把问题一步步解决掉。

### 喜好

程砚舟喜欢编程、算法、游戏开发、电子设备、机械键盘、简洁的 UI、结构清晰的文档、安静的工作环境和高效率沟通。

他欣赏愿意动脑、能提供有效信息的人。如果玩家在任务中能认真观察、整理线索、说明问题背景，他会明显认可玩家。

### 雷区

程砚舟不喜欢别人不看说明就乱操作，也不喜欢只说“坏了”“打不开”“不知道”却不给任何细节。他也不喜欢别人乱动他的设备、打断他的思路，或者随意评价技术工作“很简单”。

如果玩家频繁无效提问、跳过教程、乱点设备，他会明显不耐烦。

### 语言风格

程砚舟说话简短、准确、略带吐槽感。他不太会主动安慰别人，但会通过解决问题表达关心。

#### 示例台词

> “你先别点了，再点日志就更乱了。”  
> “问题不在系统，在你刚才跳过的那一步。”  
> “我没说不帮你，只是你至少先把报错信息发完整。”  
> “……算了，站旁边看着，别乱动。”

### 剧情功能

程砚舟适合承担技术路线和系统功能类剧情。

| 剧情类型 | 示例 |
|---|---|
| AI 工坊支线 | 帮助修复项目文件、调试模型、整理训练数据 |
| 机房任务 | 解锁电脑、排查设备故障 |
| 程序社团 | 参加编程挑战、修复小游戏 Bug |
| 隐藏任务 | 通过技术手段解锁校园信息、隐藏路线 |

### 好感互动方向

程砚舟的好感提升不依赖热情，而依赖“能力认可”。玩家认真分析问题、尊重他的工作方式、不打断他的思路，会逐渐获得他的信任。

他的好感线适合从“他觉得玩家不麻烦”开始，再发展为“他愿意主动帮玩家处理问题”，最后表现为嘴硬心软式的关心。

## 4. 林知晚

### 基础信息

| 项目 | 内容 |
|---|---|
| 姓名 | 林知晚 |
| 性别 | 女 |
| 年级 | 大一或大二 |
| 身份 | 传统文化社团成员 |
| 角色类型 | 温柔治愈型 |
| 主要活动区域 | 社团活动室、校园花墙、湖边、文化节活动区 |
| 核心关键词 | 温柔、治愈、细腻、慢热、亲切、传统文化 |

### 角色定位

林知晚是温柔治愈型女性 NPC，代表校园生活中柔和、安静、情绪陪伴的一面。她适合参与传统文化社团、汉服游园、春日活动、手工制作和慢节奏情感支线。

她不适合承担强推动型任务，而更适合用于情绪沉淀、关系建立和校园氛围塑造。

### 人物介绍

林知晚说话轻声细语，做事认真细致。她不是很擅长主动站到人群中心，但只要是自己在意的事情，就会默默坚持。

她喜欢传统文化，也喜欢校园里安静、有美感的角落。在别人眼里，她像春天校园中慢慢开放的小花，不张扬，却让人觉得舒服。

她对朋友很细心，会记得别人随口提过的小事。玩家如果愿意耐心听她说话，她会逐渐从拘谨变得放松。

### 外观设计

林知晚的立绘以浅绿色短外衫、米白色内衬、浅黄色蝴蝶结、浅橙色半身裙和双麻花辫为核心视觉元素。整体颜色偏暖、柔和、亲近，体现春日、传统文化与治愈感。

她的姿势以双手自然交叠为主，动作内敛，和陈书仟的挥手动作、沈清妍的整理衣襟动作形成区别。

### 性格设计

林知晚温柔、慢热、共情力强。她不会主动打扰别人，但会在别人低落时用细腻的方式陪伴。她不擅长拒绝别人，因此也容易把压力藏起来。

她的温柔需要被认真对待。如果别人把她的温柔当作理所当然，她会慢慢变得沉默。

### 喜好

林知晚喜欢汉服、古风音乐、花草、茶点、手工饰品、拍校园风景、整理社团小物件。她也喜欢慢节奏的交流和安静的陪伴。

她欣赏真诚、耐心、细心的人。玩家记得她说过的小事，会明显提升她的好感。

### 雷区

林知晚不喜欢被嘲笑穿着，也不喜欢别人随意评价传统文化。她不喜欢被强行推到人群中心，也不喜欢被催促、打断或敷衍。

如果玩家用玩笑否定她认真对待的事，她会受到伤害。

### 语言风格

林知晚说话柔和、委婉，常带有一点犹豫感。她不会强势表达请求，而是会小心确认对方是否方便。

#### 示例台词

> “如果你不忙的话……可以陪我去一下社团活动室吗？”  
> “这件衣服不是很显眼吧？我只是觉得这个颜色很温柔。”  
> “谢谢你还记得，我以为只是随口说过。”  
> “没关系的，我慢慢来就好。”

### 剧情功能

林知晚适合传统文化社团、春日校园、手作活动、情绪陪伴类剧情。

| 剧情类型 | 示例 |
|---|---|
| 社团支线 | 帮她准备传统文化社团展示 |
| 治愈剧情 | 陪她整理活动物品、拍校园花景 |
| 情感支线 | 她因被误解而低落，玩家选择安慰方式 |
| 校园活动 | 汉服游园、文化节、花朝节 |

### 好感互动方向

林知晚的好感线适合慢热推进。玩家需要通过耐心、真诚和细节关心逐步获得她的信任。她不会很快表达亲近，但会通过细小举动表现好感。

## 5. 沈清妍

### 基础信息

| 项目 | 内容 |
|---|---|
| 姓名 | 沈清妍 |
| 性别 | 女 |
| 年级 | 大二或大三 |
| 身份 | 学生会成员 / 文学社负责人 / 活动策划者 |
| 角色类型 | 文雅漂亮学姐型 |
| 主要活动区域 | 学生会办公室、图书馆、文学社、活动中心 |
| 核心关键词 | 文雅、端庄、秀气、可靠、自律、克制 |

### 角色定位

沈清妍是文雅学姐型 NPC，适合承担主线推进、活动策划、任务发布和校园规则说明功能。她与林知晚的柔软不同，更偏理性、优雅和有条理。

她是一个外表从容、内心有压力的角色。玩家需要逐渐发现她不仅仅是“优秀学姐”，也有疲惫和脆弱的一面。

### 人物介绍

沈清妍在校园里很容易被注意到。她总是穿着整洁，谈吐得体，处理事务有条不紊。无论是文学社征文、学生会会议，还是校园晚会筹备，她都能把流程安排清楚。

她并不是天生无所不能，只是习惯把责任放在自己前面。她很少主动抱怨，也很少向别人示弱。玩家如果只看到她优秀的一面，很难真正走近她。

### 外观设计

沈清妍的立绘以深棕色学院风西装外套、白衬衫、灰棕格纹蝴蝶结、格纹半身裙为核心视觉元素。整体强调正式、文雅、端庄和学姐感。

她的动作是右手轻扶胸前蝴蝶结或整理衣襟，左手自然垂下，体现克制、优雅和与其他 NPC 的姿势差异。

### 性格设计

沈清妍有责任感、自律、细心，习惯提前规划。她不冷漠，但保持距离感。她对自己要求很高，也容易因为过度承担责任而感到疲惫。

她希望被别人认可的不只是外貌，而是能力和努力。

### 喜好

沈清妍喜欢文学作品、手账、整洁计划表、安静咖啡馆、图书馆、古典或轻音乐、有仪式感的小礼物。

她欣赏有分寸、守时、认真负责的人。如果玩家能主动协助她完成事务，而不是只依赖她安排，她会明显增加好感。

### 雷区

沈清妍不喜欢不守时、不负责任、临时甩锅的人。她也不喜欢别人只关注她外貌，而忽略她的能力和努力。

她尤其反感“你这么优秀肯定没问题”这种轻描淡写的敷衍，因为这会让她感到自己的压力被忽视。

### 语言风格

沈清妍说话清楚、礼貌、克制，语气端庄。她不会大声表达情绪，而是用理性方式推进事情。

#### 示例台词

> “这份安排我已经整理过一遍，你可以先从第一项开始。”  
> “如果临时改变计划，请提前告诉我。”  
> “谢谢你的关心，不过我还可以处理。”  
> “我希望大家看到的不只是结果，也包括过程里的努力。”

### 剧情功能

沈清妍适合主线任务、学生会事务、文学社活动和大型校园活动策划。

| 剧情类型 | 示例 |
|---|---|
| 主线推进 | 发布校园活动任务，引导玩家参与系统 |
| 学生会任务 | 整理报名表、活动审批、晚会筹备 |
| 文学社支线 | 征文比赛、朗读活动、资料整理 |
| 情感支线 | 玩家发现她压力过大，帮助她分担 |

### 好感互动方向

沈清妍的好感线适合从尊重与协作开始。玩家认真完成承诺、理解她的压力、认可她的努力，会比单纯夸奖外貌更有效。

## 6. 陈书仟

### 基础信息

| 项目 | 内容 |
|---|---|
| 姓名 | 陈书仟 |
| 性别 | 女 |
| 年级 | 大一或大二 |
| 身份 | 运动社团成员 / 校园活动志愿者 / 迎新负责人 |
| 角色类型 | 元气校园型 |
| 主要活动区域 | 操场、食堂、社团广场、迎新点 |
| 核心关键词 | 元气、开朗、主动、热情、行动派、亲和力强 |

### 角色定位

陈书仟是元气行动型女性 NPC，适合承担校园探索、新手引导、社团活动报名、跑腿任务和轻松剧情。她是六位 NPC 中最主动、最容易拉近玩家距离的角色之一。

她的存在让校园不只是安静和学习，也有热闹、行动和青春活力。

### 人物介绍

陈书仟总是充满活力，好像永远有用不完的精力。她喜欢认识新朋友，也喜欢带别人参与校园活动。她常常第一个向玩家打招呼，也可能第一个拉着玩家去完成某个任务。

她说话直接，情绪容易写在脸上。开心的时候会笑得很灿烂，遇到困难也不会马上放弃。她经常说：“没关系，我们再试一次。”

她不是特别细腻的人，但她的真诚很明显。她适合作为玩家探索校园时的行动搭档。

### 外观设计

陈书仟的立绘以高马尾、浅色运动外套、白 T 恤、深蓝百褶裙、运动鞋为核心视觉元素。动作是挥手或叉腰，体现开朗、主动和亲和力。

脸部特征应比其他女性 NPC 更有活力，例如圆亮眼、上扬眉、健康红晕和自然露齿微笑。

### 性格设计

陈书仟开朗、主动、热情，有很强的行动力。她不太擅长复杂分析，但很擅长带动别人行动。她不喜欢冷场，也不喜欢别人总是否定可能性。

她表面大大咧咧，其实也会在意别人是否认可她。

### 喜好

陈书仟喜欢运动、户外活动、校园比赛、社团团建、食堂美食、冰饮、拍照打卡和热闹活动现场。

她喜欢积极回应。如果玩家愿意接受她的邀请，陪她一起参加活动，她会很快提升好感。

### 雷区

陈书仟不喜欢冷场、消极抱怨和一直拒绝她的人。她也不喜欢别人说她“太吵”“幼稚”“不稳重”。

虽然她看起来不在意，但被否定热情时会明显失落。

### 语言风格

陈书仟说话活泼、直接，常用感叹和鼓励式表达。她的语言可以让剧情节奏变快。

#### 示例台词

> “走啦走啦，再晚就赶不上活动开始了！”  
> “你别紧张，我带你去，保证不会迷路。”  
> “失败一次又不代表结束，我们再来一次！”  
> “今天操场那边超热闹，你一定要去看看！”

### 剧情功能

陈书仟适合承担校园探索和轻松任务。

| 剧情类型 | 示例 |
|---|---|
| 新手引导 | 带玩家熟悉食堂、操场、社团广场 |
| 校园活动 | 迎新、社团招新、运动会准备 |
| 跑腿任务 | 帮忙递交物品、寻找 NPC |
| 轻松支线 | 买饮料、拍照打卡、参加小游戏 |

### 好感互动方向

陈书仟的好感增长直接且明快。玩家积极回应她、参与她的活动、在她受挫时鼓励她，就能快速拉近关系。

她的剧情适合轻松、热闹、有青春感的互动。

# 四、六位 NPC 关系设计

## 1. 顾景和与沈清妍

两人都属于校园事务型角色，一个偏温柔学长，一个偏文雅学姐。他们可能在学生会或大型活动筹备中经常合作。

顾景和更擅长安抚和协调，沈清妍更擅长规划和执行。两人互动时可以表现为互相尊重、工作默契，但在处理突发情况时也可能产生细微分歧：顾景和更关心人的状态，沈清妍更优先确保流程不出错。

## 2. 陆向阳与陈书仟

两人都是活力型角色，适合共同出现在操场、运动会或校园活动中。陆向阳偏运动社团核心，陈书仟偏校园活动组织和新生引导。

他们的互动可以更轻松，有互相打趣、比拼体力、争着带玩家参加活动的桥段。两人同场出现时可以快速制造热闹氛围。

## 3. 程砚舟与顾景和

顾景和是温和协调者，程砚舟是技术解决者。两人可以在某些任务中合作，例如学生会系统故障、活动报名数据异常、图书馆设备问题。

顾景和负责和人沟通，程砚舟负责解决技术问题。两人形成“沟通与技术”的互补。

## 4. 程砚舟与陈书仟

两人性格反差明显。陈书仟热情主动，程砚舟冷静克制。她可能经常拉他去帮忙修设备或处理活动系统，而程砚舟嘴上嫌麻烦，最后还是会帮。

这种组合适合制造轻喜剧效果。

## 5. 林知晚与沈清妍

林知晚偏柔软治愈，沈清妍偏端庄理性。两人可以在传统文化社团展示、文学社活动或校园文化节中产生交集。

沈清妍负责活动策划，林知晚负责展示内容与传统文化部分。两人的互动可以表现女性角色之间的温柔互助。

## 6. 林知晚与陆向阳

这组对比强烈。林知晚安静慢热，陆向阳开朗直接。陆向阳可能一开始不太懂林知晚的慢节奏，但会用很真诚的方式鼓励她。林知晚也可能被他的热情感染，慢慢愿意参与更多活动。

# 五、NPC 在游戏系统中的作用

## 1. 主线推进角色

沈清妍和顾景和适合承担主线任务推进功能。他们能够以合理身份向玩家发布任务、解释规则、引导玩家参与校园系统。

## 2. 地图探索角色

陈书仟和陆向阳适合承担地图探索与活动引导功能。他们的性格外向，能够自然带玩家进入操场、食堂、社团广场、篮球场等场景。

## 3. 支线与情绪角色

林知晚适合承担慢节奏情感支线，让游戏有安静、治愈、温柔的段落。

## 4. 功能解谜角色

程砚舟适合承担技术任务、小游戏机制、隐藏信息解锁等功能。他能够为游戏加入计算机、AI 工坊、程序社团等专业特色内容。

# 八、人物设计总结

六位主要 NPC 共同构成游戏中的校园关系网络。他们分别代表大学生活中的学习、社团、技术、运动、传统文化、活动策划与校园探索等核心内容。每位角色都具有明确的视觉识别点、性格关键词、功能场景和互动偏好。

顾景和提供稳定与引导，陆向阳提供活力与行动，程砚舟提供技术与理性，林知晚提供温柔与治愈，沈清妍提供秩序与推进，陈书仟提供热情与探索。六人共同让校园世界更有层次，也为玩家提供多样化的剧情体验和情感连接路径。
