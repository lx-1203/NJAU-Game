## 一、项目概述

### 1.1 项目基本信息

| 项目                   | 内容                              |
| ---------------------- | --------------------------------- |
| **游戏名称**     | 《钟山下》                        |
| **英文名称**     | Beneath Zhongshan                 |
| **游戏类型**     | 大学生活模拟 / 文字冒险 / 养成类  |
| **平台**         | PC（Steam）                       |
| **目标分级**     | 12+（适合青少年及以上）           |
| **预计开发周期** | 12-18个月（业余时间/兼职开发）    |
| **核心团队规模** | 3-5人（远程协作，无实体办公成本） |

### 1.2 一句话 pitch

> **"在钟山脚下的南京农业大学，度过你独一无二的大学四年——每一个选择，都将通向截然不同的人生。"**

### 1.3 核心定位

《钟山下》是一款以**真实中国大学生活**为题材的沉浸式模拟养成游戏。玩家将扮演南京农业大学的新生，在40个回合（对应大学四年）中通过数百个事件和选择，体验学业、社交、恋爱、社团、兼职等丰富的校园生活，最终导向**58个截然不同的结局**。

**产品差异化定位**：

- 首款聚焦"中国本土大学生活"的深度模拟游戏
- 真实还原南农高校生活细节（地标、食堂、校园跑、绩点系统）
- 高重玩价值：58个结局 × 多周目传承 × 随机事件

## 二、市场分析

### 2.1 市场规模与趋势

| 市场维度                | 数据与趋势                                                       |
| ----------------------- | ---------------------------------------------------------------- |
| **独立游戏市场**  | 2025年中国独立游戏市场规模预计突破80亿元，Steam中文用户占比超30% |
| **模拟养成赛道**  | 《火山的女儿》销量超100万份，验证国产养成游戏商业潜力            |
| **怀旧/校园题材** | B站"大学生活"相关内容年均播放量超50亿，用户情感共鸣强烈          |
| **女性向市场**    | 乙女游戏、情感模拟游戏用户付费率高，女性玩家占比超60%            |

### 2.2 目标用户画像

**核心用户群（18-28岁）**：

- 大学生：正在经历或即将经历大学生活，寻求共鸣与代入
- 毕业1-5年的职场新人：怀旧校园时光，弥补遗憾
- 模拟经营/养成游戏爱好者：《火山的女儿》《中国式家长》玩家

**次级用户群**：

- 高中生：对大学生活的向往与好奇
- 海外留学生：对中国校园文化的兴趣
- 南农校友：情感连接与捐赠潜力

### 2.3 竞品分析

| 竞品                     | 类型     | 优势                       | 劣势                           | 我们的机会                     |
| ------------------------ | -------- | -------------------------- | ------------------------------ | ------------------------------ |
| **《火山的女儿》** | 养成模拟 | 美术精美、情感细腻、多结局 | 西方奇幻题材、缺乏中国文化共鸣 | 本土校园题材更易引发情感共鸣   |
| **《中国式家长》** | 休闲模拟 | 话题性强、社交传播广       | 玩法相对单一、重复可玩性有限   | 更复杂的系统深度、更长线的养成 |
| **《太吾绘卷》**   | 沙盒武侠 | 系统深度极高               | 门槛高、学习成本大             | 更易上手的现代题材             |
| **《翌日》系列**   | 恐怖校园 | 校园题材                   | 类型局限、受众较窄             | 正向积极的校园体验             |

**市场空白点**：

- 中国大学生活的深度模拟游戏几乎空白
- 缺少聚焦"大学生成长抉择"的叙事游戏
- Steam平台缺乏优质中文校园题材作品

### 2.4 市场机会总结

| 机会点           | 说明                                                         |
| ---------------- | ------------------------------------------------------------ |
| **情感共鸣** | 中国大学生群体庞大（4000万+），但缺乏真正"懂他们"的游戏      |
| **怀旧经济** | 毕业人群对校园生活的怀念，形成稳定的情感付费动力             |
| **传播潜力** | B站、小红书等平台的"大学生活"话题热度高，易于病毒传播       |
| **差异化**   | 相比西方校园模拟（如《模拟人生》大学DLC），本土细节更真实    |

## 三、产品核心设计

### 3.1 核心循环

```
回合开始 → 选择行动（学习/社交/社团/兼职等）→ 触发事件 → 属性变化 → 学期末结算 → 剧情推进 → 下一回合
```

### 3.2 核心系统

#### 3.2.1 属性系统（9大属性）

| 属性类别   | 属性名称 | 作用描述               | 主要获取途径         |
| ---------- | -------- | ---------------------- | -------------------- |
| **学业属性** | 学力     | 决定考试成绩、保研资格 | 图书馆、上课、自习   |
|            | GPA      | 影响奖学金、就业       | 考试成绩换算         |
| **人际属性** | 魅力     | 影响NPC好感度          | 打扮、社团、社交活动 |
|            | 领导力   | 影响社团地位、仕途线   | 学生会、社团管理     |
| **身心属性** | 体魄     | 影响校园跑、健康       | 跑协、健身房、运动   |
|            | 心情     | 过低触发负面事件       | 娱乐、社交、休息     |
|            | 压力     | 过高触发崩溃事件       | 学习、考试、社交     |
| **隐藏属性** | 负罪感   | 影响道德选择           | 作弊、劈腿等黑暗行为 |
|            | 黑暗值   | 解锁黑暗结局           | 持续负面行为累积     |
|            | 勇气     | 影响关键抉择           | 特殊事件、挑战       |

#### 3.2.2 回合系统（40回合 = 大学四年）

| 学期       | 回合数 | 关键事件                      |
| ---------- | ------ | ----------------------------- |
| **大一上** | 1-5    | 入学、军训、百团大战、期末考  |
| **大一下** | 6-10   | 恋爱触发、专业分流、暑期实习  |
| **大二上** | 11-15  | 四六级、社团换届、第一次挂科  |
| **大二下** | 16-20  | 入党申请、暑期社会实践        |
| **大三上** | 21-25  | 考研/考公/就业方向选择        |
| **大三下** | 26-30  | 实习、保研夏令营              |
| **大四上** | 31-35  | 秋招、保研面试、考研报名      |
| **大四下** | 36-40  | 春招、毕业论文、毕业结局      |

#### 3.2.3 事件系统（~400个事件）

| 事件类型     | 数量   | 说明                                  |
| ------------ | ------ | ------------------------------------- |
| **主线事件** | ~40个  | 每回合必触发，推动学期剧情            |
| **随机事件** | ~200个 | 根据属性/好感度触发，如"偶遇NPC"     |
| **抉择事件** | ~100个 | 关键选择影响属性和剧情走向            |
| **黑暗事件** | ~15个  | 作弊/代考/灰色兼职/劈腿等道德抉择     |
| **NPC对话**  | ~200条 | 含好感度分支、专属支线                |

#### 3.2.4 结局系统（重玩核心）

**毕业结局**：52个（4个特殊+48个星级结局）

- 巅峰线：校园传说（7★）、南农之光（6★）
- 学术线：顶尖高校直博~学术梦碎（5个）
- 仕途线：选调生·基层~体制梦碎（13个，含2个性别限定、5个军旅方向）
- 创业线：获得天使投资~创业失败（5个）
- 职场线：世界500强管培生~毕业即失业（6个）
- 留学线：世界名校全奖直博~留学梦碎（5个）
- 文体线：职业运动员~健身博主（4个）
- 新兴职业：AI独角兽创始人~赛博流浪（6个）
- 黑暗线：灰色产业链老大~混子勉强毕业（5个）
- 保底线：万人迷/老好人~无业游民（3个）

**恋爱结局**：6个（甜蜜~无恋爱）

### 3.3 特色创新点

#### 3.3.1 真实校园体验

| 真实元素           | 游戏呈现                                              |
| ------------------ | ----------------------------------------------------- |
| **校园跑**     | 必须参加，体魄不够会挂科；可找代跑（被抓负罪感+20）   |
| **体测**       | BMI、引体向上/仰卧起坐，影响奖学金和保研              |
| **绩点系统**   | 真实换算：60分=1.0，90分=4.0，影响奖学金、就业        |
| **宿舍生活**   | 室友关系、夜聊、外卖、逃课点名互助                    |
| **食堂系统**   | 卫岗食堂真实还原，金钱不够只能吃泡面                  |
| **教务系统**   | 抢课、挂科、重修、清考，全流程模拟                    |

#### 3.3.2 情感叙事

- **NPC情感深度**：12个可恋爱角色，每个有独立剧情线和多结局
- **毕业情感**：结局时触发"毕业纪念册"，回顾四年选择
- **遗憾与弥补**：多周目传承系统，继承部分属性重新来过

## 四、商业模式

### 4.1 收入模式

| 模式     | 说明                     | 预期占比 |
| -------- | ------------------------ | -------- |
| **买断制** | Steam/移动端一次性购买   | 90%      |
| **DLC**  | 新学校、新结局、新NPC    | 8%       |
| **内购** | 皮肤、头像框（可选）     | 2%       |

**定价策略**：

- 标准版：¥58（Steam）、¥30（移动端）
- 折扣期：¥38（EA阶段）
- 免费试玩（大一上5回合）+ 完整版解锁（¥30）
- 去广告内购（可选）

### 4.2 成本估算（AI极致降本版）

> **核心理念**：穷书生做学问，一分钱掰成两半花。美术、音乐全面AI化，人力靠热爱发电，不拿投资也能活。

| 成本项               | 金额（万元）      | 说明                                     |
| -------------------- | ----------------- | ---------------------------------------- |
| **人力成本**   | 30-50             | 3-5人×18个月，兼职/远程为主，基本生活费 |
| **AI工具订阅** | 3-5               | Midjourney、ChatGPT、Suno等AI订阅费用    |
| **外包成本**   | 5-10              | 少量关键素材人工精修（主视觉、LOGO）     |
| **测试发行**   | 5-8               | Steam上架费、小范围测试推广              |
| **运营成本**   | 2-3               | 服务器（云服务新人包）、社区运营         |
| **预留资金**   | 10                | 风险缓冲                                 |
| **总计**       | **55-86万** | 不足传统独立游戏成本的10%                |

**降本策略详解**：

- **美术AI化**：角色立绘、场景背景、事件CG均由Midjourney/Stable Diffusion生成，策划自主修图。无需专职美术，省去外包费50-80万
- **音乐AI化**：BGM、音效使用Suno AI、Udio等工具生成，根据场景情绪实时调整。省去原创音乐成本20-30万
- **程序自研**：使用开源框架，拒绝昂贵商业插件
- **云资源薅羊毛**：充分利用AWS/阿里云学生包、免费额度
- **社区共建**：核心玩家参与测试反馈，UGC内容补充

### 4.3 盈利预测（零投资压力版）

> **不需要投资人，不需要FA，不需要路演。卖一份赚一份，活着就是胜利。**

**活下去底线**：

- Steam销量：1万份 × ¥48 × 70%（平台分成后）= 33.6万
- **即可覆盖全部开发成本**（按55万计，首月回款+分期）

**小富即安**：

- Steam销量：3万份 × ¥48 × 70% = 100.8万
- 人均分20万（5人团队），够付两年房租 + 下一款游戏启动资金

**意外之喜**：

- Steam销量：10万份（参考《火山的女儿》1/10）= 336万
- 移动端/主机移植 = 额外收入
- **全员可以全职做游戏了**

**盈亏平衡点**：仅需 **8000份销量** 即可回本（按55万成本计）

> 对比传统独立游戏（成本500万+），我们的风险系数降低90%。即使失败，也只是几个月的业余时间，而不是倾家荡产。

| 事件CG   | Midjourney v6                 | 分镜脚本、角色一致性控制         |
| BGM      | Suno AI / Udio                | 情绪标签输入、片段剪辑拼接       |

**预期产出效率**：

- 美术资产：AI生成1张立绘/场景 < 10分钟，单人日产出30-50张，筛选后可用10-15张
- 音乐资产：AI生成一首BGM < 5分钟，整游戏BGM库（30首）可在1-2天内完成初版
- 文案产出：AI辅助下，单人日均可完成10-15个事件脚本初稿

### 5.4 风险评估与对策（低成本容错版）

> **最大优势：成本低，输得起。最坏结果也就是浪费一年时间，不会负债不会跑路。**

| 风险                         | 等级 | 对策                                                |
| ---------------------------- | ---- | --------------------------------------------------- |
| **内容量过大**         | 中   | MVP只做大一上（5回合），验证好玩再继续；不好玩就止损 |
| **AI美术质量不稳定**   | 中   | 多生成多筛选；关键素材人工精修；接受"AI味"美学风格 |
| **题材局限性**         | 低   | 中国大学生这么多，哪怕只打动1%也是大市场 |
| **团队流失**           | 中   | 兼职开发，无全职绑定；文档完善，随时可交接 |
| **销量不及预期**       | 低   | 成本极低，8000份就回本；EA模式，边卖边做 |
| **AI工具政策变化**     | 低   | 多工具备份（MJ/SD/Leonardo），避免单一依赖 |

**底线思维**：

- 如果6个月后MVP不好玩 → 果断放弃，各自安好，无债务纠纷
- 如果EA销量惨淡 → 砍掉后续内容，就当完成一个作品，积累经验
- 如果有人退出 → 文档齐全，随时可招新人或缩小规模继续

## 七、成功标准（穷书生版）

> 不追求暴富，只求证明一件事：**几个普通人用AI工具，也能做出好游戏。**

### 7.1 活下去的目标

| 指标                | 底线目标   | 开心目标   | 做梦目标   |
