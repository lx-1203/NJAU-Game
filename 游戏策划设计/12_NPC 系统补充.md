### 12.1 好感度系统

#### 12.1.1 好感度等级

| 等级 | 范围   | 称号      | 解锁内容                                           | 回合结算效果                       |
| ---- | ------ | --------- | -------------------------------------------------- | ---------------------------------- |
| Lv.0 | 0~19   | 陌生人    | 仅「聊天」选项                                     | 无                                 |
| Lv.1 | 20~39  | 点头之交  | 「一起吃饭」解锁                                   | 无                                 |
| Lv.2 | 40~59  | 朋友      | 「帮助对方」「送礼物」解锁；解锁 NPC 背景故事片段  | 偶尔收到 NPC 消息（纯叙事）        |
| Lv.3 | 60~79  | 好友      | 专属支线事件解锁；「约会」选项解锁；NPC 会主动找你 | 每回合有 30% 概率触发 NPC 主动事件 |
| Lv.4 | 80~99  | 挚友      | 「告白」选项解锁；深层支线事件                     | 每回合有 50% 概率触发 NPC 主动事件 |
| Lv.5 | ≥ 100  | 灵魂伴侣  | 解锁专属结局剧情；NPC 特殊互动                     | 每回合有 60% 概率触发 NPC 主动事件，额外心情+2 |

#### 12.1.2 好感度计算公式

```
实际好感度变化 = 基础值 × 魅力系数 × 性格匹配系数 × 状态惩罚系数 × 重复衰减系数

其中：
  魅力系数     = 1 + 魅力值 / 100
  性格匹配系数 = 偏好互动 ×1.3~1.5 / 中性 ×1.0 / 减益 ×0.7~0.9
  状态惩罚系数 = 由负罪感/摆烂值决定（见下表）
  重复衰减系数 = 同一行动连续执行时的递减
```

**状态惩罚系数表：**

| 状态       | 区间                      | 好感度惩罚系数 |
| ---------- | ------------------------- | -------------- |
| 正常       | 负罪感 < 60 且摆烂值 < 60 | × 1.0         |
| 中度负罪感 | 60 ≤ 负罪感 < 90         | × 0.75        |
| 重度负罪感 | 负罪感 ≥ 90              | × 0.5         |
| 中度摆烂   | 60 ≤ 摆烂值 < 90         | × 0.7         |
| 重度摆烂   | 摆烂值 ≥ 90              | × 0.4         |
| 多重叠加   | 取最低系数                | 取最低值       |

**重复衰减系数：**

| 同一行动对同一 NPC 连续执行次数 | 衰减系数 |
| ------------------------------- | -------- |
| 第 1 次                         | × 1.0   |
| 第 2 次（连续）                 | × 0.8   |
| 第 3 次（连续）                 | × 0.5   |
| 第 4 次及以上（连续）           | × 0.3   |
| 间隔 1 回合后重置计数           | —       |

> **设计意图：** 防止玩家无脑刷同一行动堆好感度，鼓励多样化社交。

#### 12.1.3 好感度自然衰减

| 关系等级       | 每回合自然衰减 | 说明                     |
| -------------- | -------------- | ------------------------ |
| 陌生人         | 0              | 不衰减                   |
| 点头之交       | -1             | 轻微衰减                 |
| 朋友           | -2             | 不联系就会疏远           |
| 好友           | -3             | 好友需要维护             |
| 挚友           | -4             | 深度关系更需要投入       |
| 灵魂伴侣(≥100) | -6             | 最高级关系，维护成本最高 |
| 恋人（已告白） | -5             | 恋爱关系需要大量精力维系 |

> **好感度无上限：** 好感度可超过100继续增长，但超过100后每回合衰减加速（-6/回合），需要持续高投入才能维持。

##### 2. 外貌描述

**女版（苏晚宁）：**
身高163cm，体型偏瘦但不单薄，常年端正坐姿让她仪态格外好。齐肩中短发，日常用一只深棕色发夹别在耳后，露出光洁饱满的额头。圆框眼镜后面是一双偏细长的杏眼，笑起来弯成月牙但不常笑。肤色偏白，不怎么晒太阳的那种白。穿衣风格极其固定：秋冬是奶白色针织衫配深色直筒裤，春夏换成浅色棉麻衬衫。书包里永远插着一支黑色斑马牌中性笔。给人的第一印象是"安静的、不太好接近的好学生"，但仔细看会发现TA的衣服袖口总沾着一小块墨渍。

**说话风格：**
语速偏慢，措辞谨慎，习惯把话说完整而不是用缩写或网络用语。很少用感叹号，表达情绪的方式是沉默的时长变化——高兴的时候话会稍微多一两句，不高兴就整个人安静下来。发微信喜欢打句号，几乎不用表情包（好感度高了之后偶尔会发一个小猫的表情包，对TA来说已经是极限了）。从不主动聊八卦，但如果你说了TA会认认真真听完。学术相关话题是唯一能让TA变得健谈的领域，一旦聊到专业内容语速会明显加快。

##### 4. 背景故事

苏晚宁/陆知行出生在江苏南通的一个普通教师家庭。父亲是高中化学老师，母亲是小学语文老师，家里没有大富大贵但教育氛围浓厚。从小到大TA都是"别人家的孩子"那类角色——成绩稳定在年级前十，不惹事不张扬，老师提起来总说"这孩子省心"。

但"省心"这个词，其实是TA心里一根很细的刺。

小时候TA其实也贪玩过，想学画画、想养一只猫、想周末不做卷子出去骑车。但父母总是温和而坚定地说："先把该做的事做完，再说别的。"他们不是虎爸虎妈，不打不骂，只是把"优秀=安全"这个等式刻进了TA的骨子里。于是TA学会了一种本领：**把所有不安、渴望和犹豫都折叠起来，塞进课本的夹层里。**

高考时TA考了全省前5000名。本来可以冲更高的学校，但填志愿那天晚上TA独自查了很久，最终把南京农业大学的生物科学填在了第一志愿。父母有些意外，TA只说："离家不算远，这个专业我喜欢。"没说出口的理由是——高二那年暑假TA在学校生物实验室看到显微镜下洋葱表皮细胞的那一刻，第一次觉得有什么东西不是因为"应该"而是因为"想要"。

来南农之后，TA的生活轨迹极其规律：宿舍-图书馆-实验室，三点一线。成绩稳居专业前三，拿了两次校级奖学金。但TA的朋友很少——不是不想交，是不太会。TA的善意永远表达得很笨拙：看见室友在写论文，TA会默默把自己整理好的参考文献清单放到对方桌上，然后假装什么都没做。**TA的内心矛盾在于：TA知道自己的世界太小了，也隐约渴望某种更热烈、更不确定的东西，但TA害怕一旦踏出那个安全的圈，就不再是"那个靠谱的人"了。**

TA的手机相册里有一个加密文件夹，里面全是TA偷偷拍的校园风景——钟山的晚霞、图书馆窗台上的落叶、食堂阿姨多给的一块红烧肉。这是TA从未让任何人看过的、属于自己的那一面。

##### 5. 好感度各阶段解锁的背景故事片段

**Lv.2「初识」-> 图书馆的秘密**

> 一次你在图书馆三楼找位置，发现TA坐的桌角贴了一张小小的便利贴，上面用极细的笔迹写着："今天窗外的梧桐叶掉了第一片。9.14。"你后来翻TA的桌面，发现类似的便利贴有厚厚一沓，全是些无用但细腻的记录。TA发现你在看，脸一下子红了，手忙脚乱把便利贴全收进了文具盒："这个……就是随便写写的，没什么意义。"但TA没有生气。

**Lv.3「走近」-> 那通电话**

> 某天深夜你在阳台吹风，听见TA在楼梯间打电话，声音很低但能听清："妈，我知道，我会考虑读研的……不是不想回去，我就是……"沉默了很久。"我知道你们是为我好。"挂掉电话后TA靠在墙上站了一会儿，抬头看见你，愣了几秒，然后露出一个很勉强的微笑："没事，家里催着做规划呢。"那天晚上TA没有像往常一样11点准时关灯。

**Lv.4「交心」-> 加密相册**

> TA终于打开了手机里那个加密文件夹给你看。几百张照片全是南农的日常——清晨空无一人的操场跑道上结的霜、实验室窗台上不知谁放的多肉植物、你在食堂笑着说什么但TA从后面偷偷拍下的侧脸。"其实我一直在看这些，"TA的声音很轻，"以前觉得只要自己记得就好了，但现在……好像有点想跟你一起看。"

##### 4. 背景故事

江晓萱/方屿洋来自安徽合肥，家境中上——父亲经营一家中型建材公司，母亲是三甲医院的护士长。小时候家里条件最好的那几年，父亲忙于应酬，母亲忙于夜班，TA在小学三年级之前几乎是被外婆带大的。外婆家在合肥老城区，弄堂里邻居来来往往，打招呼、串门、蹭饭是日常。**TA的社交天赋不是天生的，是在那条弄堂里"学"来的——TA很早就发现，只要自己笑着主动说话，大人们就会喜欢自己、夸自己、不会把自己一个人丢在家里。**

初中的时候父母终于接TA回了身边。但"回家"反而让TA不适应——家里太安静了，父亲在书房接电话谈生意，母亲下了班只想安静休息。TA试过在饭桌上分享学校的事，得到的回应通常是"嗯"和"好好学习"。**TA后来总结出一个从没对人说过的道理：热闹是要自己去找的，等别人给你，会等到失望。**

高考发挥还算正常，报志愿时TA选了南京——"合肥太熟了没意思，南京够大够热闹"。来南农之后一周之内加了十几个社团的群，两周之内认识了半层楼的人，一个月后成为年级里"有事找TA准没错"的万能联络人。

但没有人注意到TA手机从不静音这件事。也没有人注意到TA从不主动关掉宿舍的灯——TA怕安静，更怕黑。TA社交能力极强，但通讯录里有三百多个好友、没有一个TA会在凌晨两点情绪崩溃的时候拨出去的号码。**TA的内心矛盾在于：TA用热闹和社交填满了所有时间缝隙，是因为一旦停下来，那种小时候被独自留在家里的空旷感就会回来。TA害怕的不是孤独本身，而是发现"原来热闹过后还是孤独"。**

##### 4. 背景故事

许逸舟/林栖晚出生在江苏淮安一个普普通通的家庭。父亲在区政府的档案科工作，母亲是银行柜员，家里经济不差也不算宽裕——用他/她自己的话说，"饿不着，但零花钱也不会多给"。

然而很少有人知道，许逸舟/林栖晚曾经并不是现在这副模样。初中时期，他/她曾是班上前五名的常客，尤其偏爱写作和绘画，笔记本上画满了密密麻麻的分镜草稿——那时候的梦想是做一个漫画家。然而中考前夕，父亲翻到了那些画稿，沉默了很久，只说了一句："画这些能考上高中吗？"画稿被收走了，他/她也没有反抗。高中三年，成绩从前列滑到中游，不是学不会，而是慢慢发现"努力了也不一定有用，那不如不努力"。

高考成绩不上不下，选南农的理由在志愿表上写的是"专业前景好"，实际原因是——"离家近，分数线刚好够，而且听说南农食堂特别好吃"。入学后很快成为宿舍里最安静的存在：不社交、不竞争、不焦虑，每天打打游戏、刷刷视频、准时去食堂吃饭。

他/她把"佛系"活成了铠甲。但在宿舍角落的抽屉最深处，有一个没上锁的铁盒子，里面放着初中时期的几张画稿——那是唯一没被收走的，因为他/她偷偷藏下来的。深夜熄灯以后，偶尔会在被窝里用手机备忘录写一些没头没尾的短故事，写完也不给任何人看。

**TA的内心矛盾：不是不想努力，是怕努力了还是没用，那就连最后的借口都没了。**

*专业：食品科学与工程——"至少毕业以后不会饿死，而且实验课能试吃。"*

##### 5. 好感度各阶段解锁的背景故事片段

**Lv.2「打开抽屉」**

> 你帮他/她从食堂带了一份糖醋排骨，随口说了句"知道你懒得下楼"。他/她愣了一下，接过饭盒的手指微微停顿，然后笑了一声："你是第一个不催我下楼，而是直接帮我带饭的人。"那天晚上你经过他/她的书桌，瞥见桌上摊着一张纸——上面是一幅未完成的铅笔素描，画的是宿舍窗外那棵梧桐。他/她顺着你的视线看过去，飞快地把纸翻了过去。"……别看，画得很烂。"

**Lv.3「铁盒子」**

> 某个期中考后的深夜，你发现他/她一个人坐在宿舍阳台上，膝盖上放着那个铁盒子。你在旁边坐下，没有说话。过了很久，他/她自己开口了："我以前其实挺拼的，你信吗？"铁盒子打开，里面是几张泛黄的漫画分镜稿，线条青涩却充满热情。"后来我爸说这些没用……其实他也没说错，但我就是再也画不动了。""不是不想努力，是怕努力了还是没用，那就连最后的借口都没了。"

**Lv.4「被窝里的故事」**

> 他/她犹豫了很久，终于把手机递给你，屏幕上是备忘录里的一篇短篇故事。写的是一只不会飞的鸟，住在一棵很高的树上，每天看别的鸟飞来飞去，觉得其实待在树上也很好。结尾那句话是："但它偶尔还是会梦到风的形状。"你抬头看他/她，他/她别过脸去，耳朵尖红透了。"……别笑啊，我知道写得很中二。而且这是第一次给人看。你，你是唯一一个。"

**说话风格：**

##### 4. 背景故事

顾衍之/裴知遥来自云南昆明，家中独生子/女。父亲是中科院昆明植物研究所的研究员，母亲在云南大学教授生态学。他/她从小在标本室和实验田之间长大，别的孩子在背唐诗的年纪，他/她已经能认出三十种以上的兰科植物。家庭氛围开明但学术氛围浓厚——自由是真的自由，但"优秀"是呼吸般自然的默认项。

选择南京农业大学风景园林专业，源于高中时读到一篇关于南农菊花遗传育种团队的论文。入学后迅速成为专业里的标杆：绩点常年第一，主持过一项省级大创项目，拿过"钟山学子"奖学金，是院学生会副主席。

但在光环之下，顾衍之/裴知遥有一段从不提起的过往。高二那年暑假，带他/她走上植物学启蒙路的外公在一次野外科考中意外离世。外公生前最后的研究课题是滇西北的一种濒危高山杜鹃，标本和笔记留了一半。那枚银色素戒是外公的遗物。顾衍之/裴知遥继承了那枚戒指，也继承了那份未完成的执念。

大三确定保研至中国科学院大学（国科大），方向是植物景观生态学。离别是他/她性格中"不轻易靠近"的深层原因——见过了突然的失去，就学会了控制期待。但正因如此，一旦他/她选择靠近某个人，那份重量是无法回头的。

*专业：风景园林——"人不能凌驾于自然之上，但可以学着和它对话。"*

### 12.6 告白系统（完整版）

#### 告白成功率公式

```
告白成功率 = 基础成功率 + 魅力修正 + 压力修正 + 好感修正 + 地点修正 + NPC修正

其中：
  基础成功率 = 50%
  魅力修正   = 魅力值 × 0.2%（如魅力120→+24%）
  压力修正   = -压力值 × 0.1%（如压力50→-5%）
  好感修正   = (好感度 - 80) × 1%（80以上每点+1%）
  地点修正   = 图书馆+15%, 食堂+10%, 宿舍+10%, 校庆+20%, 梧桐大道+10%, 紫金山+15%
  NPC修正    = NPC 个人告白加成（见§12.3）

成功率上限 = 95%，下限 = 20%
```

#### 告白流程

1. 玩家选择「告白」→ 选择告白地点
2. 系统播放告白场景描述（根据地点生成）
3. 玩家选择告白台词（3 选 1）：直球型 / 委婉型 / 浪漫型（纯叙事差异化，不影响成功率）
4. 系统计算成功率 → 掷骰判定
5. 成功 → NPC 回应 + 进入恋爱状态（好感 +10，心情 +15）
6. 失败 → NPC 拒绝台词 + 进入冷却期（好感 -20，心情 -8，压力 +10，冷却 3 回合）
