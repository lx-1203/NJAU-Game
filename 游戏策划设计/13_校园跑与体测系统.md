## 13. 校园跑与体测系统

> Physical Education System
> 体测和学科考试完全不同：学科考的是"知识"，用选择题；体测考的是"身体"，
> 用**策略选择+体魄值判定**——玩家不是在回答问题，而是在**做决策**。

### 13.1 体测事件

- **触发时机：** 每学期固定触发一次（期末考试周最后一项）
- **考核项目：** 3 个项目 × 3 次策略选择 = **共 9 次决策**

| 项目        | 性别差异                | 考查属性   | 玩法         |
| ----------- | ----------------------- | ---------- | ------------ |
| 🏃 中长跑   | 男1000米 / 女800米      | 体魄(核心) | 3 次策略选择 |
| 💪 力量项目 | 男引体向上 / 女仰卧起坐 | 体魄       | 3 次策略选择 |
| 🦶 立定跳远 | 无差异                  | 体魄       | 3 次策略选择 |

### 13.2 中长跑——详细设计

中长跑分为 **3 个阶段**，每个阶段让玩家做一次策略选择：

**第一阶段：起跑（前 1/3）**

| 选择            | 体力消耗 | 成绩加成 | 后续影响               |
| --------------- | -------- | -------- | ---------------------- |
| A. 匀速慢跑     | 低       | +0       | 中间阶段选择不受限     |
| B. 加速拉开距离 | 高       | +10      | 中间阶段出现"岔气"风险 |
| C. 慢慢跑/摸鱼  | 极低     | -10      | 无惩罚但成绩已落后     |

**第二阶段：途中跑（中间 1/3）** — 文字描述根据上一阶段选择动态变化

| 选择            | 效果               |
| --------------- | ------------------ |
| A. 保持当前节奏 | 稳妥，不好不坏     |
| B. 咬牙加速超人 | 拼一把，体力消耗大 |
| C. 慢下来走几步 | 恢复体力，但成绩掉 |

> **连续加速风险：** 如果第一阶段选了 B(加速) 且第二阶段又选 B(继续加速)：70% 概率成功坚持(+15)，30% 概率岔气被迫停下(-20)。

**第三阶段：冲刺（最后 1/3）**

| 选择                | 效果                   |
| ------------------- | ---------------------- |
| A. 全力冲刺         | 最后一搏，成绩加成最高 |
| B. 维持速度跑过终点 | 稳健完赛               |
| C. 走过去           | 彻底放弃               |

**策略组合示例：**

| 策略风格 | 三次选择             | 加成                   | 适合谁             |
| -------- | -------------------- | ---------------------- | ------------------ |
| 稳健型   | 匀速→保持→维持     | +0~+5                  | 体魄够高，不用冒险 |
| 冲刺型   | 匀速→保持→全力冲刺 | +10~+15                | 想拼个好成绩       |
| 赌徒型   | 加速→加速→冲刺     | +35(成功) 或 -15(失败) | 体魄不够想搏一搏   |
| 摆烂型   | 摸鱼→走几步→走过去 | -30                    | 真的不在乎         |

**中长跑成绩计算：**

```
最终得分 = 体魄值基础分 + 三次选择的累计加成

体魄值基础分：
  体魄 ≥ 120 → 基础分 80
  体魄 80~119 → 基础分 65
  体魄 50~79  → 基础分 50
  体魄 < 50   → 基础分 30

三次选择累计加成范围：-30 ~ +35
```

### 13.3 力量项目（男引体向上 / 女仰卧起坐）

同样分 **3 次选择**，模拟"做了一部分之后要不要继续拼"的心理：

- **第一次选择（开头）：** 标准动作稳做 / 快速拉多做几个 / 先挂一会儿
- **第二次选择（中途）：** 休息几秒再继续 / 一鼓作气 / 下来了够了
- **第三次选择（最后）：** 再拼一个 / 稳稳落地 / 求同学帮忙托一下

> **C选项（作弊·让同学托一下）：** 成功 70%（成绩+2，负罪感+3），被老师发现 30%（成绩作废重测，心情-5）。
> 体测作弊代价比考试作弊小（无"开除"风险），但与负罪感系统联动。

### 13.4 立定跳远

立定跳远给予 **3 次跳跃机会，取最好成绩**：

每次跳跃前选择：

- A. 正常起跳 → 稳定发挥，成绩 = 基础距离 ± 随机5cm
- B. 深蹲蓄力全力跳 → 70% 概率远 +15cm，30% 概率踩线犯规(0分)
- C. 随便跳一下 → 基础距离 - 20cm

```
基础距离(cm) = 体魄值 × 1.5 + 50
  体魄100 → 200cm（及格线附近）
  体魄120 → 230cm（良好）
  体魄150 → 275cm（优秀）
  体魄 60 → 140cm（不及格）
```

> 策略：第1次保守（正常跳）先拿底，第2、3次赌一把看能不能超越。

### 13.5 体测总成绩计算

```
体测总分 = (中长跑得分 + 力量项目得分 + 立定跳远得分) / 3
```

| 总分  | 等级   | 效果                              |
| ----- | ------ | --------------------------------- |
| ≥ 90 | 优秀   | 体育课加分，心情 +5               |
| 75~89 | 良好   | 正常通过                          |
| 60~74 | 及格   | 正常通过，压力 +3                 |
| < 60  | 不及格 | 体育课挂科风险，压力 +10，心情 -8 |

### 13.6 体魄值不够时的「紧急处理」

如果体魄值很低（< 50），体测开始前会触发额外选择：

| 选项                  | 效果                                                                 |
| --------------------- | -------------------------------------------------------------------- |
| A. 硬着头皮上         | 正常参加体测                                                         |
| B. 找老师说身体不舒服 | 50% 成功延期到补测；50% 被识破照常测                                 |
| C. 找人代测（50元）   | 被发现 40%（体测 0 分 + 负罪感 +10）；没发现（成绩良好 + 负罪感 +5） |

> **代测 = 体测版的"作弊"**，与负罪感和金钱系统联动。

### 13.7 体测叙事文本

体测最大特色是**叙事感**，每次选择后的反馈是画面感文字而非冷冰冰的数字：

| 场景                 | 文本示例                                                                                    |
| -------------------- | ------------------------------------------------------------------------------------------- |
| 体魄高+全力冲刺      | "你像一阵风一样冲过终点，体育老师难得地朝你点了点头。"                                      |
| 体魄低+拼命坚持      | "最后100米你几乎是用意志力在跑，冲过终点那一刻直接瘫在了草地上，但心里有一种奇怪的满足感。" |
| 体魄低+中途放弃      | "你跑到一半就开始走了，和旁边几个同学一起聊着天慢悠悠'散步'到终点。"                        |
| 引体向上一个都做不了 | "你挂在杆上拼命往上蹬，脸涨得通红，但身体纹丝不动……最后在沉默中落了地。"                  |
| 全力跳·犯规         | "用力过猛！起跳的时候脚尖滑过了线。'犯规！重来！'体育老师毫不留情。"                        |

### 13.8 体测随学期的变化

每学期体测项目相同，但叙事文本**语气随学期演变**：

| 学期   | 变化                                                                 |
| ------ | -------------------------------------------------------------------- |
| 大一上 | 第一次体测，文本偏"紧张、新鲜"。"你第一次站在大学的跑道上…"         |
| 大一下 | 有经验了，文本偏"熟练"。"又是体测，你已经知道套路了。"               |
| 大二上 | 体测标准略微提高（及格线+5分）。"大二的标准比去年高了一点…"         |
| 大二下 | 最后一次强制体测，文本偏"感慨"。"最后一次体测了，跑道还是那条跑道。" |

### 13.9 校园跑行动

- **每学期目标次数：** 32 次（若加入体育社团则维持 32 次；若无体育社团则需额外 +20 次，共 52 次）
- **每次效果：** 体魄 +1，行动点 -1
- **代跑价格：** 5~8 元/次，越靠近期末越贵
- **玩家可自行安排**每回合跑几次，灵活分配至学期的 5 个回合内
- **累计跑步次数**影响学期体测成绩
