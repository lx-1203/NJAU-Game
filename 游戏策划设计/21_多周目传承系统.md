## 21. 多周目传承系统

> New Game Plus System
> 
> **设计理念：** 每一次通关都是对"钟山下"的一次更深理解。多周目不是简单的数值堆叠，而是解锁被隐藏的可能性——包括那些被首周目社会规训所遮蔽的真实自我。

---

### 21.1 传承触发条件

**基础条件：** 完成任意结局（包括坏结局）即可触发传承

**特殊说明：**
- 坏结局（一跃解千愁/网贷破产/半途而废/学术不端·开除）同样可触发传承，但传承系数×0.7
- 正常结局传承系数×1.0
- 7星/6星巅峰结局传承系数×1.3

---

### 21.2 可传承内容

#### 21.2.1 初始属性加成

根据上周目最终属性，按以下公式计算本周目初始加成：

```
本周目初始属性 = 基础初始值 + min(上周目终值 × 传承比例, 上限值)

传承比例：
  - 首周目→二周目：10%
  - 二周目→三周目：15%（累计约23.5%）
  - 三周目→四周目：20%（累计约38.8%）
  - 四周目及以上：25%（累计约54.1%，之后边际递减）

上限值：
  - 学力/魅力/体魄：首周目100，每多一周目+20，最高200
  - 领导力/勇气/幸运：首周目30，每多一周目+10，最高80
```

**属性传承示例（以学力为例）：**

| 周目 | 上周目终值 | 传承比例 | 获得加成 | 本周目初始学力 |
|------|-----------|---------|---------|---------------|
| 首周目 | — | — | — | 100（基础） |
| 二周目 | 180 | 10% | +18 | 118 |
| 三周目 | 220 | 15% | +33 | 133 |
| 四周目 | 260 | 20% | +52 | 152 |

---

#### 21.2.2 初始好感度

**传承规则：**
- 上周目所有已认识NPC（好感度≥20）的好感度×20%继承
- 上周目已攻略成功的恋人：好感度×30%继承，且初始关系等级为"好友"
- 新周目中与这些NPC的首次互动有额外剧情（"我们是不是在哪见过？"）

**好感度上限：**
- 继承的好感度最高不超过60（防止二周目直接告白）
- 超过60的部分转化为"羁绊值"，用于解锁特殊对话

---

#### 21.2.3 初始金钱

**计算公式：**

```
本周目初始金钱 = 8000（基础）+ max(上周目剩余金钱 × 5%, 0) + 周目奖励

周目奖励：
  - 二周目：+500
  - 三周目：+1000
  - 四周目及以上：+1500

最高上限：15000元
```

**说明：** 
- 只有"剩余金钱"（通关时拥有的钱）才能传承，已花费的不会返还
- 坏结局通关时金钱为负数的，按0计算（不会扣减本周目初始金钱）

---

#### 21.2.4 初始天赋点

**天赋点获取：**
- 每通关一次，根据结局星级获得天赋点

| 结局星级 | 获得天赋点 |
|---------|-----------|
| 7星（校园传说） | 10点 |
| 6星（南农之光） | 8点 |
| 5星 | 6点 |
| 4星 | 4点 |
| 3星 | 3点 |
| 2星 | 2点 |
| 1星 | 1点 |
| 坏结局 | 0点（但解锁周目） |

**天赋树结构：**

```
                    【起点】
                       │
        ┌──────────────┼──────────────┐
        │              │              │
    【社交系】      【学业系】      【生存系】
        │              │              │
   ┌────┴────┐    ┌────┴────┐    ┌────┴────┐
   │         │    │         │    │         │
 魅力+5   口才  学力+5   记忆力  金钱+200  抗压
 (1点)   (2点)  (1点)   (2点)   (1点)   (2点)
   │         │    │         │    │         │
   └────┬────┘    └────┬────┘    └────┬────┘
        │              │              │
     【共情】        【洞察】        【直觉】
     (3点)          (3点)          (3点)
        │              │              │
        └──────────────┼──────────────┘
                       │
                 【真我觉醒】
                   (5点)
              解锁隐藏剧情线
```

**具体天赋效果：**

| 天赋名称 | 消耗 | 效果 | 解锁条件 |
|---------|------|------|---------|
| **魅力+5** | 1点 | 初始魅力+5 | 无 |
| **口才** | 2点 | 告白成功率+10%，班委竞选+10% | 魅力≥60 |
| **共情** | 3点 | 所有NPC好感度增长速度+20% | 已选"口才" |
| **学力+5** | 1点 | 初始学力+5 | 无 |
| **记忆力** | 2点 | 考试通过率+10%，复习效果+15% | 学力≥60 |
| **洞察** | 3点 | 可看穿NPC隐藏属性（显示好感度具体数值） | 已选"记忆力" |
| **金钱+200** | 1点 | 初始金钱+200 | 无 |
| **抗压** | 2点 | 压力增长-20%，心情恢复+10% | 体魄≥60 |
| **直觉** | 3点 | 解锁隐藏事件触发预警（回合开始时提示） | 已选"抗压" |
| **真我觉醒** | 5点 | 解锁隐藏剧情线"第二人生" | 已选共情/洞察/直觉任一 |

### 21.3 周目限定解锁内容

#### 21.3.1 二周目解锁

| 解锁内容 | 说明 |
|---------|------|
| **同性恋攻略线** | 可攻略同性别NPC（室友A/B/C），新增专属剧情与结局 |
| **隐藏角色·转校生** | 大二上学期转入的转校生NPC，只在二周目及以上出现 |
| **前世记忆碎片** | 特定回合触发"既视感"事件，暗示上周目的选择 |
| **速通模式** | 可开启"跳过已读剧情"功能 |

#### 21.3.2 三周目解锁

| 解锁内容 | 说明 |
|---------|------|
| **NPC全记忆** | 已攻略过的NPC会保留上周目记忆，有特殊对话 |
| **隐藏结局·时间循环** | 特定条件触发"困在同一天"的meta结局 |
| **自定义开局** | 可自由分配初始属性点（上限受传承限制） |

#### 21.3.3 四周目及以上

| 解锁内容 | 说明 |
|---------|------|
| **真结局线** | 解锁"钟山之下"真结局，需满足苛刻条件 |
| **开发者注释** | 游戏中显示策划注释，解释设计意图 |
| **自由模式** | 无回合限制，可无限探索 |

### 21.5 数据存储结构

```csharp
[System.Serializable]
public class NewGamePlusData
{
    public int currentPlaythrough;           // 当前周目数（1=首周目）
    public int totalTalentPoints;            // 累计获得天赋点
    public int spentTalentPoints;            // 已花费天赋点
    public List<string> unlockedTalents;     // 已解锁天赋ID列表
    
    // 上周目数据（用于计算传承）
    public float lastPlayFinalScholarship;   // 上周目学力终值
    public float lastPlayFinalCharm;         // 上周目魅力终值
    public float lastPlayFinalFitness;       // 上周目体魄终值
    public float lastPlayFinalLeadership;    // 上周目领导力终值
    public float lastPlayFinalCourage;       // 上周目勇气终值
    public float lastPlayFinalLuck;          // 上周目幸运终值
    
    public Dictionary<string, int> lastPlayNpcAffection;  // 上周目NPC好感度
    public int lastPlayFinalMoney;           // 上周目剩余金钱
    public EndingType lastPlayEnding;        // 上周目结局类型
    public int lastPlayEndingStars;          // 上周目结局星级
    
    // 周目解锁标记
    public bool lgbtUnlocked;                // 同性恋线已解锁
    public bool transferStudentUnlocked;     // 转校生已解锁
    public bool trueEndingUnlocked;          // 真结局线已解锁
}
```

### 21.6 同性恋攻略线设计

> **解锁条件：** 二周目及以上自动解锁

#### 21.6.1 可选对象

| NPC | 性别限定 | 特点 | 新增剧情 |
|-----|---------|------|---------|
| **室友A（温暖学霸）** | 男/女皆可 | 学术型，内向 | 图书馆偶遇深化为深夜谈心 |
| **室友B（社交达人）** | 男/女皆可 | 社交型，外向 | 酒吧微醺后的真心话 |
| **室友C（佛系玩家）** | 男/女皆可 | 游戏宅，随性 | 深夜双排时的告白 |

#### 21.6.2 特殊机制

- **社会压力值**：同性恋线新增隐藏属性，高时触发父母质问、同学议论等事件
- **隐藏选项**：部分社交场合有"装作普通朋友"和"坦然承认"的选择
- **专属结局**：6个同性恋专属恋爱结局 + 2个隐藏社会压力结局

#### 21.6.3 叙事处理

- 不做任何道德评判，仅呈现大学生真实处境
- 社会压力事件可通过高魅力/高领导力化解
- 坏结局包括"被迫分手""地下恋情疲惫"等

### 21.7 与其他系统的联动

| 系统 | 联动方式 |
|------|---------|
| **结局系统** | 多周目结局增加"传承值"评级，影响总评分 |
| **成就系统** | 新增"轮回者"系列成就（通关3次/5次/10次） |
| **钟山台调试** | 可模拟任意周目状态进行测试 |
| **新闻系统** | 高周目时校园新闻会出现"既视感"彩蛋 |

### 21.8 MVP裁剪方案

**首周目不涉及，二周目MVP最小实现：**

- [ ] 基础属性传承（10%）
- [ ] 同性恋线解锁（仅室友A）
- [ ] 天赋树基础框架（仅3个1点天赋）

**后续版本扩展：**

- [ ] 完整天赋树
- [ ] 好感度传承
- [ ] 转校生NPC
- [ ] 真结局线
- [ ] 周目专属成就

---

> **设计意图：** 多周目传承不是为了"变强"，而是为了"看见"——看见首周目没能走的那条路，看见被隐藏的自己，看见这个游戏全部的温柔与残酷。
