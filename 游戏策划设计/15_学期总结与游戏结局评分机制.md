## 15. 学期总结与游戏结局评分机制

> Semester Summary & Ending Evaluation System

### 15.1 学期评分触发时机

- **触发时机：** 每学期第五回合（期末结算后）自动触发
- **表现形式：** 弹出成绩单界面，展示本学期四项维度得分及总分，评级后存档

### 15.2 学期评分计算

学期总分 = 学业分 + 人际分 + 体育分 + 特殊成就分 − 扣分项

各维度详见 [§6.4 期末总结属性](#64-期末总结属性影响结局)

**学期评级：**

| 总分区间       | 评级        | 说明                               |
| -------------- | ----------- | ---------------------------------- |
| ≥ 6000 分     | **S** | 完美学期，解锁专属剧情或成就       |
| 4500 ~ 5999 分 | **A** | 优秀，下学期部分属性有加成         |
| 3000 ~ 4499 分 | **B** | 良好，正常推进                     |
| 1500 ~ 2999 分 | **C** | 一般，下学期初始压力 +5            |
| < 1500 分      | **D** | 较差，下学期初始压力 +10，心情 -10 |

**评级附加效果：**

| 评级 | 下学期初始加成               |
| ---- | ---------------------------- |
| S    | 学力 +10，心情 +10，压力 -10 |
| A    | 学力 +5，心情 +5             |
| B    | 无额外效果                   |
| C    | 压力 +5                      |
| D    | 压力 +10，心情 -10           |

**评级解锁内容：**

| 评级            | 解锁内容                                                        |
| --------------- | --------------------------------------------------------------- |
| S               | 解锁该学期专属隐藏支线剧情；累计两个 S 解锁特殊成就「学霸人生」 |
| A               | 解锁 NPC 专属对话一条                                           |
| D（连续两学期） | 触发「颓废警告」事件，有转机选项                                |

### 15.3 游戏结局综合评分

**触发时机：** 第八学期（大四下）第五回合结束后

**计算方式：**

```
最终综合分 = Σ（各学期总分 × 学年权重）
```

| 学年             | 权重   |
| ---------------- | ------ |
| 大一（学期 1~2） | × 1.0 |
| 大二（学期 3~4） | × 1.5 |
| 大三（学期 5~6） | × 2.0 |
| 大四（学期 7~8） | × 2.5 |

**人生规划方向加权：**

在"事件 10：人生规划"中选择的目标方向，对应维度额外 × 1.5 计入最终综合分：

| 选择方向    | 额外加权维度                     |
| ----------- | -------------------------------- |
| 保研 / 考研 | 学业分 × 1.5                    |
| 考公        | 学业分 × 1.2，特殊成就分 × 1.3 |
| 创业        | 特殊成就分 × 1.5                |
| 进企业      | 人际分 × 1.5                    |
| 出国留学    | 学业分 × 1.3，特殊成就分 × 1.2 |
| **军旅生涯** | **体育分 × 2.0，特殊成就分 × 1.2** |
| 听天由命    | 无额外加权                       |

### 15.4 结局判定逻辑

游戏结局由**毕业结局**和**恋爱结局**两部分组成，各自独立判定后同屏并列呈现。

```
游戏结局 = 毕业结局（必有，1枚）+ 恋爱结局（条件触发，0~1枚）
```

**星级体系（1~7星）：** 每个毕业结局均标注星级，代表该结局的综合成就高度。

| 星级                  | 含义 | 条件强度概述                                           |
| --------------------- | ---- | ------------------------------------------------------ |
| ★★★★★★★（7星） | 传说 | 全属性极高，GPA 顶尖，负罪感极低，多项行为计数全部达标 |
| ★★★★★★（6星）   | 辉煌 | 全属性很高，GPA 优秀，负罪感低，综合分极高             |
| ★★★★★（5星）     | 卓越 | 主线属性≥200，GPA≥3.5 或专项能力极高                 |
| ★★★★（4星）       | 优秀 | 主线属性≥160，GPA≥3.0 或专项能力较高                 |
| ★★★（3星）         | 良好 | 主线属性≥120，有一定专项积累                          |
| ★★（2星）           | 及格 | 主线属性≥100，满足基础门槛                            |
| ★（1星）             | 保底 | 无特殊要求，兜底结局                                   |

星级越高，结局描述文本越长、越具叙事感，解锁的 CG 与成就也越丰富。

#### 15.4.1 判定优先级流程图

毕业结局在**第八学期第五回合结束后**按以下优先级从上到下逐层检查，命中即停：

```
第0层 ── 特殊/强制结局（一跃解千愁、网贷破产、半途而废、学术不端·开除）
            │  ← 中途或结算时触发，优先级最高
            ↓ 未触发
第1层 ── 巅峰结局（7星 校园传说 → 6星 南农之光，从高到低检查）
            │  ← 全属性极高 + 全方位发展的「真结局」
            ↓ 不满足
第2层 ── 人生规划方向结局（按玩家在「事件10：人生规划」中选择的方向，
            │   检查对应类型的结局：5星→4星→3星→2星→1星）
            │   【学术/考研】→【仕途/考公】→【创业】→【职场】→【留学】→【军旅生涯】
            ↓ 不满足
第3层 ── 非规划方向结局（玩家实际行为偏离规划时，检查其他类型结局）
            ↓ 不满足
第4层 ── 黑暗结局（黑暗值 ≥ 60 时进入此层检查）
            ↓ 不满足
第5层 ── 文体/特长结局
            ↓ 不满足
第6层 ── 新兴职业/自由职业结局（AI画师、游戏主播、独立开发者等数字时代新赛道）
            ↓ 不满足
第7层 ── 保底结局（万人迷·女/老好人·男 → 普通毕业生 → 无业游民）
```

> **说明：** 同一层内从高星级向低星级逐条检查，命中第一个即为最终毕业结局。

#### 15.4.2 特殊/强制结局（第0层，优先级最高）

特殊结局在游戏过程中**实时监测**，一旦满足条件立即触发，跳过后续所有判定。

| 网贷破产       | —   | 金钱 < −5000 元                                                                 | 手机上跳出第 47 条催收短信的时候，你已经不会害怕了。退学手续办得很快，快到你还没来得及想，行李就已经被搬出了宿舍。欠下的数字还在涨，而你连回家的路费都凑不出来。                                     |

> **设计意图：** 这四个结局不参与星级评定（标记为"—"），代表不可挽回的人生岔路。它们的存在是为了让玩家意识到：心理健康、财务自律、学术诚信与坚持本身，是比任何成就都重要的底线。

#### 15.4.5 仕途结局（考公/考编/参军方向，**10 个结局**）

> 以领导力为核心驱动属性，学力、GPA 为基础，入党、班委等社会资本为关键分流条件，负罪感的约束比学术线更严格（体现体制内对"干净履历"的隐性要求）。含 2 个性别限定结局（征兵入伍·男 / 女兵·军旅青春·女）。

**【军旅生涯专属机制】**

> 当玩家在「事件 10：人生规划」中选择「军旅生涯」方向后，解锁以下特殊机制：

**1. 入伍报名审核系统（大四上学期触发）**

- **触发时机：** 第七学期第一回合（大四上 9 月）自动触发
- **触发条件：** 人生规划选择 = 军旅生涯

**审核项目及淘汰机制：**

| 审核项 | 通过标准 | 未通过后果 |
| ------ | -------- | ---------- |
| **学业审查** | 累计挂科次数 = 0 且 作弊被抓次数 = 0 | 直接失去入伍资格，跳转至「军旅梦碎」结局线 |
| **体魄审查** | 体魄 ≥ 180 | 进入「体能特训」事件链，有 2 回合时间通过锻炼弥补 |
| **政审初审** | 黑暗值 ≤ 40 且 负罪感 ≤ 60 | 进入「政审辅导」事件，需完成指定任务降低负面属性 |
| **心理评估** | 心理健康值 ≥ 50 | 进入「心理调适」事件，需进行心理咨询 |

> **设计说明：** 学业审查是硬门槛（无补考机会），体魄审查有补救窗口，体现现实中征兵对"无违纪记录"的严格要求。

**2. 体能特训事件链（体魄未达标时触发）**

若体魄 < 180，触发「加练」选项：

| 特训项目 | 行动点消耗 | 体魄增益 | 成功率 |
| -------- | ---------- | -------- | ------ |
| 晨跑加练 | 2 | +3~5 | 90% |
| 健身房集训 | 3 + 金钱 200 | +5~8 | 85% |
| 退役学长指导 | 2 + 需认识「体育类 NPC」 | +6~10 | 95% |
| 户外拉练（紫金山） | 3 | +4~6 | 80% |

- 特训持续 2 回合，若 2 回合后体魄仍 < 180，失去本年度入伍资格
- 若选择放弃特训，体魄 < 180 也可继续游戏，但结局最高只能达到 3 星

**3. 入伍志愿选择（审核通过后触发）**

| 志愿类型 | 体魄要求 | 额外加成 | 结局星级上限 |
| -------- | -------- | -------- | ------------ |
| 特种部队 | ≥ 250 | 勇气 +10 | 5 星 |
| 技术兵种 | ≥ 200 且 学力 ≥ 180 | 学力 +5 | 5 星 |
| 普通义务兵 | ≥ 180 | 无 | 4 星 |
| 军校报考 | ≥ 200 且 GPA ≥ 3.0 且 学力 ≥ 200 | 领导力 +5 | 5 星 |

#### 15.4.15 核心规律总结

##### 星级与条件强度对照

| 星级 | 主属性要求    | GPA 要求          | 专项/行为计数 | 负罪感要求 |
| ---- | ------------- | ----------------- | ------------- | ---------- |
| 7星  | 全属性 ≥ 250 | ≥ 3.9            | 多项全部达标  | ≤ 15      |
| 6星  | 全属性 ≥ 200 | ≥ 3.7            | 多项达标      | ≤ 20      |
| 5星  | 主属性 ≥ 200 | ≥ 3.5 或专项极高 | 有严格要求    | ≤ 30~40   |
| 4星  | 主属性 ≥ 160 | ≥ 3.0 或专项较高 | 有中等要求    | ≤ 50~60   |
| 3星  | 主属性 ≥ 120 | ≥ 2.5            | 有基础要求    | 视线路     |
| 2星  | 主属性 ≥ 100 | 无硬性要求        | 仅需方向匹配  | 视线路     |
| 1星  | 无要求        | 无要求            | 无要求        | 无要求     |

##### 负罪感分界线（纯 debuff 属性）

> 负罪感不直接决定结局方向，但通过效率惩罚将玩家从高星推向低星。

| 负罪感区间 | 定位 | 效果概述                                     |
| ---------- | ---- | -------------------------------------------- |
| ≤ 20      | 清澈 | 无任何负面效果，可冲击巅峰结局（6~7星）      |
| 21 ~ 50    | 中性 | 轻微效率衰减，不影响 3~5 星结局              |
| 51 ~ 89    | 内耗 | 正向行动收益打折，行动点额外消耗，限制高星   |
| 90 ~ 149   | 崩溃 | 正向行动大幅锁定，核心属性持续衰减           |
| ≥ 150     | 临界 | 全正向行动锁定，极速触发抑郁，摆烂值持续攀升 |

##### 黑暗值分界线（黑暗结局驱动属性）

> 黑暗值不产生 debuff，只是结局方向的分流器。

| 黑暗值区间 | 定位 | 可达结局类型                    |
| ---------- | ---- | ------------------------------- |
| ≤ 59      | 正途 | 不进入黑暗结局判定层            |
| 60 ~ 79    | 灰色 | 黑暗结局入门（混子·勉强毕业）  |
| 80 ~ 99    | 暗影 | 中位黑暗结局（PUA、代考、诈骗） |
| ≥ 100     | 深渊 | 高位黑暗结局（灰色产业链老大）  |

##### 摆烂值分界线（退学驱动属性）

> 摆烂值与学力相反，代表玩家放弃学业的程度。

| 摆烂值区间 | 定位     | 效果概述                             |
| ---------- | -------- | ------------------------------------ |
| ≤ 29      | 正常     | 无影响                               |
| 30 ~ 59    | 轻度摆烂 | 学力增长打 8 折，每回合心情 +2       |
| 60 ~ 89    | 中度摆烂 | 学力增长打 5 折，旷课不再增加负罪感  |
| 90 ~ 119   | 重度摆烂 | 触发学业警告，正向行动额外消耗行动点 |
| ≥ 120     | 彻底摆烂 | 强制触发「半途而废」退学结局         |

##### 核心设计理念

- **高星结局 = 高属性 + 专项深耕 + 自律（低负罪感）**，三者缺一不可
- **三条负面属性各司其职**：负罪感 = 效率 debuff（影响"做得好不好"）；摆烂值 = 退学驱动（影响"还在不在"）；黑暗值 = 方向分流（影响"走哪条路"）
- **人生规划方向优先但不锁死**：玩家选了考研但实际属性更适合创业，系统会在第3层判定中给出创业结局，体现「人生不一定按计划走」
- **每条线都有保底**：选了某个方向的玩家至少能拿到该方向的 1 星结局，不会完全落空
- **六条毕业路线完整覆盖**：学术 5 个、仕途 **10 个**（含 2 个性别限定 + **军旅生涯 7 个**）、创业 5 个、职场 6 个（含 1 个性别限定）、留学 5 个、文体 4 个、新兴职业 6 个、黑暗 5 个、保底 3 个，每条线均覆盖 1~5 星
- **恋爱结局独立于毕业结局**：两者并列展示，互不干扰
