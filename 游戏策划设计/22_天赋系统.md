## 22. 天赋系统

> Talent System

---

### 21.1 天赋点概述

#### 什么是天赋点

**天赋点（Talent Point, TP）** 是角色成长过程中获得的珍贵资源，用于在天赋树中激活各种被动增益效果。天赋点代表玩家在大学生活中积累的「顿悟」与「成长」，是对玩家积极行为的奖励。

#### 天赋点获取途径

| 获取方式               | 天赋点奖励      | 说明                                |
| ---------------------- | --------------- | ----------------------------------- |
| **绩点奖励**     | 每学期 GPA × 2 | 每学期结算时发放，GPA 4.0 获得 8 点 |
| **学业突破**     | +3/次           | 单门课程首次达到 90 分              |
| **竞赛获奖**     | +5~10           | 校级+5，省级+8，国家级+10           |
| **四六级通过**   | 四级+3，六级+5  | 仅限首次通过                        |
| **体测优秀**     | +3/学期         | 体测达到优秀等级                    |
| **特殊成就**     | +2~5            | 完成特定成就事件                    |
| **人生规划达成** | +10             | 完成大一/大二/大三规划目标          |

#### 天赋点获取示例

```
大一上学期结算：
- GPA 3.5 → 获得 7 天赋点
- 体测良好（未达优秀）→ 0 天赋点
- 四级通过 → 3 天赋点
- 学期总计：10 天赋点

大学四年预估总获取：60~100 天赋点（视玩家表现）
```

### 21.2 天赋树结构

#### 四大天赋分支

天赋树分为四大分支，分别对应大学的四个核心维度：

```
                    【天赋树总览】
                         │
    ┌────────────┬───────┴───────┬────────────┐
    │            │               │            │
  学业线      社交线          体魄线        心境线
 (智识)      (人脉)          (意志)        (心态)
    │            │               │            │
   4层         4层             4层          4层
  共12点      共12点          共12点       共12点
```

#### 分支解锁规则

1. **默认状态**：四大分支第一层全部开放，可直接投入天赋点
2. **层数解锁**：某分支投入 N 点天赋后，解锁该分支第 N+1 层
3. **跨分支限制**：各分支独立计算，互不影响
4. **重置机制**：每学期结算后，可花费 500 金钱重置一次天赋（退还全部天赋点）

### 21.3 学业线天赋（智识）

> 提升学习效率、考试表现、科研能力

#### 第一层：基础学业（每点消耗 1 天赋点）

| 天赋名称           | 效果描述                      |
| ------------------ | ----------------------------- |
| **专注阅读** | 图书馆自习获得的学力 +15%     |
| **课堂笔记** | 上课获得的学力 +15%           |
| **考前突击** | 期末复习事件的通过率加成 +10% |

#### 第二层：进阶学业（每点消耗 2 天赋点）

| 天赋名称           | 效果描述                        |
| ------------------ | ------------------------------- |
| **触类旁通** | 自习时相邻学科同步获得 30% 学力 |
| **记忆强化** | 考试系统答题容错次数 +1         |
| **时间管理** | 自习行动点消耗 -1（最低 1）     |

#### 第三层：学术精英（每点消耗 3 天赋点）

| 天赋名称           | 效果描述                                 |
| ------------------ | ---------------------------------------- |
| **竞赛天赋** | 参加学科竞赛时，成功率 +20%              |
| **科研直觉** | 解锁「联系导师」特殊选项，可提前参与科研 |
| **绩点守护** | 期末考试时，挂科率额外 -15%              |

#### 第四层：学术大师（每点消耗 4 天赋点）

| 天赋名称           | 效果描述                                               |
| ------------------ | ------------------------------------------------------ |
| **保研预感** | GPA 排名进入前 10% 时自动提示，保研相关事件成功率 +25% |
| **学术传承** | 担任学习委员或助教时，额外获得领导力 +2/回合           |

**学业线总计**：1×3 + 2×3 + 3×3 + 4×2 = **26 天赋点**

### 21.7 天赋点消耗汇总

| 分支   | 层数   | 每点消耗 | 点数    | 小计 |
| ------ | ------ | -------- | ------- | ---- |
| 学业线 | 1~4 层 | 1/2/3/4  | 3/3/3/2 | 26   |
| 社交线 | 1~4 层 | 1/2/3/4  | 3/3/3/2 | 26   |
| 体魄线 | 1~4 层 | 1/2/3/4  | 3/3/3/2 | 26   |
| 心境线 | 1~4 层 | 1/2/3/4  | 3/3/3/2 | 26   |

**满级总计**：26 × 4 = **104 天赋点**

**设计意图**：大学四年正常游戏可获得 60~100 天赋点，玩家必须做出取舍，无法点满全部天赋。支持多种build路线：

- 学霸型：主点学业线 + 辅助体魄线
- 社交型：主点社交线 + 辅助心境线
- 均衡型：四线平均投入，走多面手路线
- 极限型：单线点满，成为某一领域的大师

### 21.8 天赋系统与属性系统联动

#### 天赋对属性的间接影响

天赋系统不直接修改基础属性值，而是通过以下方式影响：

```
【属性获取公式】
实际获得值 = 基础获得值 × (1 + 天赋加成%) × 其他修正

【成功率公式】
实际成功率 = 基础成功率 + 天赋加成 - 压力惩罚 + 幸运加成 - 负罪感惩罚
```

#### 天赋与负面属性的对抗

| 负面属性     | 克制天赋                     | 效果                       |
| ------------ | ---------------------------- | -------------------------- |
| 负罪感       | 问心无愧、自我和解、心如止水 | 降低增长速度/减轻惩罚      |
| 摆烂值       | 佛系心态、压力转化           | 降低增长速度/转化负面效果  |
| 压力         | 意志坚定、压力转化、铁人意志 | 降低增长速度/转化/提高上限 |
| 心理健康下降 | 自我调节、心理疏导、抗逆能力 | 减缓下降/自动恢复/免疫事件 |

### 21.9 UI 设计建议

#### 天赋树界面布局

```
┌─────────────────────────────────────────────────────────┐
│  【天赋树】                              剩余天赋点: 12  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│    [学业]      [社交]      [体魄]      [心境]          │
│      ▼           ▼           ▼           ▼             │
│                                                         │
│   ○ ○ ○      ○ ○ ○      ○ ○ ○      ○ ○ ○   ← 第一层  │
│   │ │ │      │ │ │      │ │ │      │ │ │              │
│   ○ ○ ○      ○ ○ ○      ○ ○ ○      ○ ○ ○   ← 第二层  │
│   │ │ │      │ │ │      │ │ │      │ │ │              │
│   ○ ○ ○      ○ ○ ○      ○ ○ ○      ○ ○ ○   ← 第三层  │
│   │ │ │      │ │ │      │ │ │      │ │ │              │
│   ○   ○      ○   ○      ○   ○      ○   ○   ← 第四层  │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  [选中天赋详情]                                         │
│  名称: 专注阅读    消耗: 1 天赋点                       │
│  效果: 图书馆自习获得的学力 +15%                        │
│  状态: 未激活 [激活]                                    │
└─────────────────────────────────────────────────────────┘
```

#### 视觉提示

- **未激活**：灰色空心圆
- **可激活**：高亮空心圆，悬停显示详情
- **已激活**：实心彩色圆（学业蓝、社交粉、体魄红、心境绿）
- **已激活但有依赖**：显示连接线

### 21.10 实现优先级

| 阶段         | 内容           | 说明                     |
| ------------ | -------------- | ------------------------ |
| **P0** | 天赋点获取机制 | 与绩点系统、成就系统联动 |
| **P0** | 第一层天赋实现 | MVP 阶段开放基础天赋     |
| **P1** | 2~3 层天赋实现 | 完整版开放进阶天赋       |
| **P1** | 天赋树 UI      | 独立界面，清晰展示       |
| **P2** | 第四层天赋     | 大师级天赋，高难度目标   |
| **P2** | 天赋重置功能   | 每学期一次重置机会       |
