## 9. UI 布局思路

> UI Layout Approach

```
┌──────────────────────────────────────────────────┐
│  [顶栏] 当前时间：大一上 · 回合2 · 10月          │
│          金钱：￥1488    行动点：●●●○○          │
├─────────────┬────────────────────────────────────┤
│             │                                    │
│  角色立绘   │      地图 / 当前场景               │
│  + 属性条   │  （点击地点触发对应事件）           │
│  （左侧栏） │                                    │
│             │                                    │
├─────────────┴────────────────────────────────────┤
│  [底栏] 行动按钮：自习 ｜ 社交 ｜ 出校门 ｜ 睡觉  │
└──────────────────────────────────────────────────┘
```

- 左侧栏显示：角色立绘、学力/魅力/体魄/领导力/压力/心情/负罪感/摆烂值/黑暗值属性条
- 顶栏显示：当前学年学期、回合数、月份、金钱余额、剩余行动点
- 中央地图：可点击各地点（宿舍/食堂/图书馆/操场等）触发对应事件
- 底栏行动按钮：快捷触发当前可用的自由行动

### 9.2 加载界面 UI 设计

> 场景切换、游戏启动等需要加载资源时展示的全屏加载页面，兼具氛围营造与玩家引导功能。

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  ┌───────┐                                   ┌───────┐  │
│  │ LOGO  │                                   │ v1.0  │  │
│  └───────┘                                   └───────┘  │
│                                                         │
│                                                         │
│                                                         │
│               ┌─────────────────────┐                   │
│               │                     │                   │
│               │    全 屏 插 画      │                   │
│               │   (教室场景背景)     │                   │
│               │                     │                   │
│               │                     │                   │
│               └─────────────────────┘                   │
│                                                         │
│                                                         │
│                                                         │
│          ┌─────────────────────────────────┐             │
│          │   💡 "正在翻找课本..."           │             │
│          └─────────────────────────────────┘             │
│                                                         │
│     ░░░░░░░░░░░░░░░░████████████░░░░░░░░░░░░░░░░░      │
│     ╰──────────── 进度条 (67%) ──────────────────╯      │
│                                                         │
│  ┌──────────────┐                     ┌──────────────┐  │
│  │ 📖 加载提示   │                     │  点击跳过 ▶  │  │
│  └──────────────┘                     └──────────────┘  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**界面元素说明：**

| 元素         | 位置         | 说明                                                           |
| ------------ | ------------ | -------------------------------------------------------------- |
| LOGO         | 左上角       | 游戏或工作室 Logo，保持品牌一致性                              |
| 版本号       | 右上角       | 淡显小字，如 `v1.0`，便于测试与反馈定位                      |
| 全屏插画     | 中央主体     | 与当前加载场景相关的插画（如教室、图书馆、操场等），增强沉浸感 |
| 加载提示文案 | 插画下方居中 | 💡 趣味性提示语，随机轮播，贴合校园生活主题                    |
| 进度条       | 提示文案下方 | 显示当前加载进度百分比，给予玩家明确的等待预期                 |
| 📖 加载提示  | 左下角       | 展示游戏小贴士、校园冷知识等，缓解等待焦虑                     |
| 点击跳过 ▶  | 右下角       | 加载完成后出现，允许玩家主动进入下一界面                       |

**加载提示文案池（随机轮播）：**

| 文案                     | 场景关联   |
| ------------------------ | ---------- |
| 💡 "正在翻找课本..."     | 通用       |
| 💡 "食堂阿姨正在打饭..." | 食堂场景   |
| 💡 "图书馆占座中..."     | 图书馆场景 |
| 💡 "正在铺床单..."       | 宿舍场景   |
| 💡 "辅导员正在点名..."   | 教室场景   |
| 💡 "快递小哥正在派件..." | 快递站场景 |

**交互逻辑：**

- 加载过程中，进度条实时更新，提示文案每 3~5 秒随机切换
- 插画可根据即将进入的场景动态匹配（如进入图书馆则显示图书馆插画）
- 加载完成后，进度条填满，"点击跳过 ▶" 按钮高亮显示
- 点击跳过按钮 → 渐隐过渡 → 进入目标场景
- 若玩家未操作，加载完成后 2 秒自动跳转
