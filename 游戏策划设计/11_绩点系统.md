## 11. 绩点系统

> GPA System

### 11.1 课程通过率计算

每门课的期末通过率由以下因素累积决定：

| 影响因素                      | 通过率变化 |
| ----------------------------- | ---------- |
| 初始通过率                    | 50%        |
| 自习该科（每次）              | +10%       |
| 上课出勤率高（全勤）          | +15%       |
| 旷课超过 3 次                 | -20%       |
| 期末突击复习（消耗 3 行动点） | +20%       |

### 11.2 分数与绩点换算

通过后，根据通过率高低映射至最终分数（60~100分区间）：

| 分数区间       | 对应 GPA  |
| -------------- | --------- |
| 90 分及以上    | 4.0       |
| 80 ~ 89 分     | 3.0       |
| 70 ~ 79 分     | 2.0       |
| 60 ~ 69 分     | 1.0       |
| 不及格（< 60） | 0（挂科） |

### 11.3 学期绩点计算

学期 GPA = 所有已修课程 GPA 的加权平均值，权重为课程学分。
