# 随机事件事件池文案

> 适用项目：校园角色扮演游戏  
> 事件用途：用于日常探索、地点触发、月度随机事件、NPC偶遇与成长节点补充  
> 属性体系：学业值、社交值、实践值、组织值、兴趣值、心态值、迷茫值

---

## 一、学习类随机事件

---

### 1. 老师突然点名

**事件类型：** 学习类  
**触发地点：** 教学楼  
**推荐触发时间：** 任意上课日  

#### 事件文案

今天的课堂原本和平常一样，老师在讲台上翻着课件，同学们有的认真记笔记，有的偷偷低头看手机。

就在你以为这节课会平静结束的时候，老师忽然合上点名册，抬头看向全班。

“我们来点一下名。”

教室里的空气瞬间安静了一秒。你听见后排有人小声吸了一口气，也有人开始疯狂给没来的同学发消息。

你的名字很快就被念到了。

#### 玩家选项

| 选项 | 结果 |
|---|---|
| 立刻答到，继续认真听课 | 学业值 +3，心态值 +1 |
| 答到后顺手提醒没来的同学 | 社交值 +3，学业值 +1 |
| 假装镇定，其实刚刚在走神 | 心态值 -1，迷茫值 +1 |
| 今天你本来就没来上课 | 学业值 -5，迷茫值 +3 |

### 2. 图书馆只剩一个座位

**事件类型：** 学习类  
**触发地点：** 图书馆  
**推荐触发时间：** 考试周 / 晚上  

#### 事件文案

图书馆的人比你想象中还多。

一排排座位几乎都被占满，有人埋头刷题，有人对着电脑改论文，还有人靠在椅背上短暂闭眼休息。

你绕了一圈，终于发现角落里还剩下最后一个空座位。

就在你准备走过去的时候，旁边也有一个抱着厚厚资料的同学看见了那个位置。

你们几乎同时停下脚步。

#### 玩家选项

| 选项 | 结果 |
|---|---|
| 坐下学习，今天不能再拖了 | 学业值 +4，心态值 +1 |
| 把座位让给对方 | 社交值 +3，心态值 +2 |
| 换去教学楼自习 | 学业值 +2，心态值 +1 |
| 放弃学习，回宿舍休息 | 心态值 +3，学业值 -2 |

**触发地点：** 教学楼  
**推荐触发时间：** 学期中段

教室里立刻响起一片低声哀叹。

你翻开试卷，发现题目并不算陌生，但有几道题刚好是你最近复习时跳过去的部分。

现在只能看你平时积累得怎么样了。

|---|---|
| 靠平时积累认真作答 | 学业值 +4 |
| 临时回忆课堂笔记内容 | 学业值 +2，心态值 -1 |
| 偷偷看旁边同学的答案 | 学业值 -1，心态值 -3，迷茫值 +2 |
| 放弃挣扎，随缘答题 | 学业值 -3，迷茫值 +3 |

有人已经列好了复习计划，有人刚刚开始翻开第一章，也有人看着厚厚的课本陷入沉默。

你知道，这一周会决定这个学期最后的结果。

| 选项 | 结果 |
|---|---|
| 制定计划，按科目逐个复习 | 学业值 +8，心态值 -2 |

| 只复习重点和老师划的范围 | 学业值 +3，心态值 +1 |
| 熬夜临时抱佛脚 | 学业值 +4，心态值 -6 |

### 5. 论文查重前夜

**事件类型：** 学习类  
**触发地点：** 宿舍  
**推荐触发时间：** 课程论文提交前  

#### 事件文案

晚上十一点，宿舍的灯还亮着。

你的课程论文终于写到了最后一页，但格式、引用和查重率还没有完全处理好。电脑屏幕上的文档看起来密密麻麻，越看越让人头疼。

群里不断有人发消息：

“你们查重了吗？”  
“参考文献格式到底怎么改？”  
“完了，我重复率有点高。”

你看着自己的论文，意识到今晚可能还不能睡。

#### 玩家选项

| 选项 | 结果 |
|---|---|
| 认真检查引用和格式 | 学业值 +5，心态值 -2 |
| 请同学帮忙看一遍 | 学业值 +3，社交值 +2 |
| 只改明显问题，先交再说 | 学业值 +1，迷茫值 +2 |
| 熬夜大改全文 | 学业值 +4，心态值 -6 |
| 直接拖到截止前最后一刻 | 学业值 -3，心态值 -4，迷茫值 +3 |

### 6. 小组作业分工

**事件类型：** 学习类 / 人际类  
**触发地点：** 教学楼 / 图书馆  
**推荐触发时间：** 课程项目期间  

#### 事件文案

老师布置了一个小组作业，需要几个人合作完成展示汇报。

你们围坐在教室角落，讨论选题、资料收集、PPT制作和课堂汇报的分工。有人主动承担任务，也有人低头沉默，似乎不太想参与。

小组作业最难的地方，往往不是作业本身，而是每个人的投入程度并不一样。

现在轮到你表态了。

#### 玩家选项

| 选项 | 结果 |
|---|---|
| 主动承担核心部分 | 学业值 +4，实践值 +3，心态值 -2 |
| 选择适合自己的任务 | 学业值 +3，社交值 +1 |
| 协调大家合理分工 | 组织值 +4，社交值 +3 |
| 尽量少做一点 | 学业值 -2，社交值 -2，迷茫值 +2 |
| 拖延到最后才交 | 学业值 -4，社交值 -4，心态值 -2 |

今天走进食堂的时候，你发现角落里多了一个新窗口。

窗口前排着不短的队伍，菜单牌也换成了新的样式。旁边有人端着刚买到的饭菜经过，香味让你忍不住多看了一眼。

室友在群里发消息：

“新开的窗口好像还不错，要不要试试？”

吃什么，似乎也是大学生活里每天都要认真面对的问题。

| 选项 | 结果 |
|---|---|
| 排队尝试新窗口 | 心态值 +4，兴趣值 +1 |
| 叫朋友一起尝鲜 | 社交值 +3，心态值 +3 |
| 还是选择熟悉的窗口 | 心态值 +1 |
| 因为排队太久放弃 | 心态值 -1 |
| 拍照发给朋友推荐 | 社交值 +2，兴趣值 +1 |

| 因压力太大和队友争执 | 实践值 +1，社交值 -4，心态值 -4 |
| 熬夜后及时复盘 | 实践值 +5，迷茫值 -1 |

### 39. 第一次收到感谢

**事件类型：** 成长类 / 人际类  
**触发地点：** 教室 / 社团 / 志愿活动地点  

#### 事件文案

事情结束后，对方认真地向你道谢。

“这次真的谢谢你，如果不是你帮忙，我可能做不完。”

这句话很简单，却让你有些意外。你原本只是觉得自己做了应该做的事，没想到对别人来说，真的产生了影响。

那一刻，你忽然觉得，自己的存在好像也能给别人带来一点点改变。

#### 玩家选项

| 选项 | 结果 |
|---|---|
| 微笑接受感谢 | 心态值 +4，社交值 +2 |
| 表示以后还可以互相帮忙 | 社交值 +4 |
| 觉得自己的努力有了意义 | 心态值 +5，迷茫值 -2 |
| 谦虚地说只是小事 | 社交值 +2，心态值 +2 |
| 因此更愿意帮助别人 | 教育倾向 +3，社交值 +3 |

### 40. 第一次认真思考未来

**事件类型：** 成长类 / 主线辅助类  
**触发地点：** 操场 / 图书馆 / 宿舍阳台  
**推荐触发时间：** 大二下 / 大三上  

#### 事件文案

夜色慢慢落下来，校园里的灯一盏盏亮起。

你走在熟悉的路上，忽然发现自己已经不像刚入学时那样，只想着下一节课、下一顿饭、下一次活动。

你开始想更远的事情。

毕业以后要去哪里？  
想成为什么样的人？  
是继续读书，还是去工作？  
是追求稳定，还是尝试冒险？  
是回到熟悉的地方，还是去一个更大的城市？

这些问题没有立刻给出答案，但它们第一次如此清楚地摆在你面前。

#### 玩家选项

| 选项 | 结果 |
|---|---|
| 决定认真规划未来 | 迷茫值 -5，心态值 +3 |
| 倾向继续深造 | 研究倾向 +5，学业值 +2 |
| 倾向考公考编 | 公职倾向 +5，组织值 +2 |
| 倾向实习就业 | 就业倾向 +5，实践值 +2 |
| 倾向创业或项目实践 | 创业倾向 +5，实践值 +3 |
| 还是暂时没有答案 | 迷茫值 +3，心态值 +1 |
