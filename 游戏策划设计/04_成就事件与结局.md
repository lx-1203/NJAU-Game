**焦虑系统（负罪感驱动的轻度 debuff）：**

> 焦虑是独立于抑郁的常见负面状态，由负罪感驱动。惩罚较轻但持续烦人，通过间接拉低心理健康值增加抑郁风险，但**绝不直接触发抑郁**。

| 焦虑等级           | 触发条件     | 效果                                                                                                                        | 消除方式                                      |
| ------------------ | ------------ | --------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------- |
| 无焦虑             | 负罪感 < 30  | 无                                                                                                                          | —                                            |
| **轻度焦虑** | 负罪感 30~59 | 每回合压力 +2；睡觉回复行动点 -1（睡不安稳）；偶尔弹出内心独白                                                              | 负罪感降至 30 以下自动消除                    |
| **中度焦虑** | 负罪感 60~89 | 每回合压力 +4、心情 -2；考试/竞赛成功率 -5%；**心理健康值 -2/回合**；10% 概率触发「失眠」（下回合行动点 -1）          | 负罪感降至 60 以下，或连续 2 回合进行放松活动 |
| **重度焦虑** | 负罪感 ≥ 90 | 每回合压力 +6、心情 -4；行动点上限 -1；**心理健康值 -4/回合**；25% 概率触发「焦虑发作」（该回合随机一个行动无法执行） | 负罪感降至 90 以下，或进行心理咨询            |

**抑郁系统（基于心理健康值，多因素驱动）：**

> 抑郁不再由单一数值（压力/心情）直接触发，而是由隐藏属性「心理健康值」综合驱动。心理健康值受社交孤立、学业挫败、经济压力、情感创伤、焦虑等多种因素影响。详见 §6 角色属性设计中的「心理健康值」定义。

| 心理健康值 | 状态               | 表现（玩家可见信号）                                                                              | 负面效果                                           |
| ---------- | ------------------ | ------------------------------------------------------------------------------------------------- | -------------------------------------------------- |
| 70~100     | 正常               | 无                                                                                                | 无                                                 |
| 50~69      | 亚健康             | 偶尔出现叙事文本：*"最近总觉得提不起劲……"*；校园新闻树洞出现暗示                              | 所有正向收益打**9 折**                       |
| 30~49      | **轻度抑郁** | 触发「**室友/好友关心**」事件（自救窗口）；UI 色调略微变灰                                  | 行动点上限 -1；所有正向收益打**8 折**        |
| 15~29      | **中度抑郁** | 触发「**辅导员建议心理咨询**」强制事件；随机跳过行动（*"今天什么都不想做……"*）          | 行动点上限 -2；每回合 20% 概率随机取消一个已选行动 |
| 0~14       | **重度抑郁** | 若**连续 2 回合未寻求帮助**（心理咨询/好友倾诉/家人通话），触发「**一跃解千愁**」结局 | 仅可选择睡觉、发呆；所有正向行动锁定               |

### 4.5 黑暗事件系统（黑暗值/负罪感相关行为的具体定义）

> **设计说明：** 以下事件是 §6.2 黑暗值增减表中列出的行为的**完整游戏机制定义**。每个行为都明确了触发条件、玩家选项、成功率、收益、风险和属性变化。

#### 4.5.1 考试作弊

> 在期末考试结算阶段，除正常考试外，玩家可以选择作弊来提高通过率。

- **触发时机：** 每学期回合五「期末考试结算」时，每门科目的考试界面新增选项
- **前提条件：** 无（任何玩家都可以选择作弊，但后果自负）

| 选项       | 效果             | 风险                                     |
| ---------- | ---------------- | ---------------------------------------- |
| 正常考试   | 按正常通过率结算 | 无                                       |
| 夹带小抄   | 该科通过率 +25%  | 被抓概率 = 20% + 压力×0.1 − 幸运×0.1  |
| 手机查答案 | 该科通过率 +40%  | 被抓概率 = 35% + 压力×0.15 − 幸运×0.1 |

**作弊成功时：**

- 该科通过率按加成后计算
- 黑暗值 +10
- 负罪感 +5

**作弊被抓时：**

- 该科成绩直接记为 0 分（挂科）
- 「作弊被抓次数」+1
- 黑暗值 +10
- 负罪感 +15
- 心情 -20
- 压力 +15
- **第 1 次被抓：** 触发「辅导员约谈·学术警告」强制事件，通报批评，取消本学期评优资格
- **第 2 次被抓：** 直接触发「学术不端·开除」特殊结局

#### 4.5.2 代考 / 代写

> 学力较高的玩家可能收到其他学生请求代考或代写论文的邀约，获得高额报酬，但风险极大。

- **触发时机：** 回合结算阶段概率弹出（非玩家主动选择）
- **触发条件：** 学力 ≥ 150 且已认识 NPC 数量 ≥ 3
- **出现概率：** 每回合 15%（大二起提升至 25%）；若黑暗值 ≥ 30，额外 +10%
- **首次触发：** 一个不太熟的同学私下找到你，支支吾吾地问："下周那个考试……你能不能帮个忙？"

| 选项              | 行动点消耗 | 效果                                                | 风险                                                              |
| ----------------- | ---------- | --------------------------------------------------- | ----------------------------------------------------------------- |
| 拒绝              | 0          | 无属性变化；若黑暗值 ≤ 20，心情 +3（正义感）       | 无                                                                |
| 帮忙代写作业/论文 | 2          | 金钱 +300~800（随学期递增），黑暗值 +8，负罪感 +4   | 被发现概率 10%：负罪感 +10，心情 -10                              |
| 帮忙代考          | 3          | 金钱 +1000~3000（随学期递增），黑暗值 +8，负罪感 +6 | 被发现概率 20% − 幸运×0.1：「作弊被抓次数」+1，触发学术不端风险 |

**持续机制：** 若选择帮忙 ≥ 3 次，解锁「职业化」选项——

- 每回合自动获得 1~2 单生意，无需额外行动点
- 金钱收入翻倍
- 但每回合黑暗值 +3（常态化灰色收入）
- 被发现概率每次 +2%（做得越多，暴露面越大）

#### 4.5.3 灰色兼职（诈骗群等）

- **触发条件：** 金钱 < 500 元 或 黑暗值 ≥ 20
- **出现概率：** 10%（金钱 < 0 时提升至 25%；黑暗值 ≥ 40 时提升至 30%）
- **首次触发：** 刷手机时看到一条群消息："日入 500，居家即可，有意者私"。你知道这大概率不太正规……

| 选项                          | 行动点消耗                                | 效果                                         | 风险                                                                                         |
| ----------------------------- | ----------------------------------------- | -------------------------------------------- | -------------------------------------------------------------------------------------------- |
| 无视/举报                     | 0                                         | 无属性变化；黑暗值 -1（小小的正义）          | 无                                                                                           |

| 深度参与（话术编写/流量引导） | 2                                         | 金钱 +1500~3000/回合，黑暗值 +20，负罪感 +12 | 每回合 20% 概率被警方约谈：金钱 -5000，压力 +30，心情 -25                                    |

**退出机制：** 玩家在任何阶段都可以选择「退出」，退出后：

- 黑暗值 -3（悬崖勒马）
- 不再触发该事件链的后续
- 但已累积的黑暗值和负罪感不会消失

#### 4.5.4 欺骗 NPC 获取好处

- **触发条件：** 魅力 ≥ 100 且与该 NPC 好感度 ≥ 20

**场景 1：借钱不还**

- 触发条件：NPC 好感度 ≥ 40

| ------------------ | -------------------------------------------------------------------------------------------- |
| 真的借钱并按时归还 | 好感度 +5，正常借还                                                                          |

**场景 2：编造理由请求帮助**

- 触发条件：NPC 好感度 ≥ 30
- 选项：编造困难让 NPC 帮你做事

| 选项                 | 效果                                                                                               |
| -------------------- | -------------------------------------------------------------------------------------------------- |
| 如实请求帮助         | 好感度 +5（被信任的感觉）                                                                          |
| 编造理由（夸大困难） | 获得帮助效果翻倍（如学力 +10 变为 +20），黑暗值 +5，负罪感 +3；20% 概率被识破：好感度 -20，心情 -5 |

**场景 3：利用恋爱关系获取好处**

- 触发条件：与该 NPC 处于恋爱状态
- 选项：以感情为筹码要求对方帮忙

| 选项     | 效果                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 正常请求 | 正常效果                                                               |
| 情感操控 | 效果翻倍，黑暗值 +5，负罪感 +5；30% 概率引发争吵：好感度 -15，心情 -10 |

#### 4.5.5 劈腿 / 多角恋机制

> 玩家可以在已有恋人的情况下，继续向其他 NPC 告白或暧昧。系统不会主动阻止，但会有严重后果。

- **前提条件：** 玩家已与某 NPC 进入恋爱状态

**机制说明：**

- 当玩家处于恋爱状态时，仍可与其他 NPC 互动、提升好感度、送礼物
- 当对另一个 NPC 好感度 ≥ 80 时，「告白」选项仍然可用
- 告白成功后同时拥有多个恋人，每多一个恋人：黑暗值 +8，负罪感 +6

**被发现机制：**

| 恋人数量 | 每回合被发现概率 | 被发现后果                                                                    |
| -------- | ---------------- | ----------------------------------------------------------------------------- |
| 2 人     | 8%              | 一个恋人好感度 -40，30% 概率触发分手                                          |
| 3 人     | 15%             | 所有恋人好感度 -50，50% 概率全部分手                                          |
| ≥ 4 人  | 25%             | 所有恋人好感度 -60，必定全部分手，且校园内传开：所有 NPC 好感度 -10，魅力 -20 |

**恋人之间的冲突事件：**

- 当同时拥有 ≥ 2 个恋人时，每回合有额外 5% 概率触发「两个恋人在食堂/图书馆撞见彼此」事件
- 此事件强制弹出，无法回避，玩家选项：

| 选项         | 效果                                                                      |
| ------------ | ------------------------------------------------------------------------- |
| 坦白一切     | 所有恋人好感度 -30，但黑暗值 -5（坦诚）                                   |
| 编造理由圆谎 | 50% 概率糊弄过去：无属性变化，黑暗值 +3；50% 概率被戳穿：同「被发现后果」 |
| 装傻         | 所有恋人好感度 -10，压力 +10，黑暗值 +5                                   |

### 4.7 入党系统

> 入党是一个跨越多个学期的渐进式事件链，不是单次选择即可完成。

- **启动时机：** 大一下学期回合七（五四青年节活动期间）或大二上学期回合十二可提交入党申请
- **入口位置：** 特定回合弹出「是否提交入党申请书？」选项

#### 入党流程阶段

| 阶段                | 所需条件                                             | 所需回合数                            | 每回合效果                                   | 行动点消耗             |
| ------------------- | ---------------------------------------------------- | ------------------------------------- | -------------------------------------------- | ---------------------- |
| 1. 提交入党申请书   | 无                                                   | 1 回合                                | 负罪感 ≤ 50，心情 +2                        | 0                      |
| 2. 入党积极分子学习 | 申请书已提交                                         | 持续 4 回合（每回合选择是否参加学习） | 领导力 +1/回合，学力 +2/回合，黑暗值 -1/回合 | 1/回合                 |

### 4.8 竞赛系统

> 竞赛是学术结局线和巅峰结局的核心驱动行为之一。

- **解锁时机：** 大一下学期回合八起可参加校级竞赛；大二上学期回合十二起可参加省级/国家级竞赛
- **入口位置：** 解锁后，特定回合弹出「竞赛报名」事件

#### 竞赛流程

每次竞赛分为三个阶段：

| 阶段            | 说明                                          | 行动点消耗    | 效果                                   |
| --------------- | --------------------------------------------- | ------------- | -------------------------------------- |
| 1. 报名         | 在竞赛报名事件中选择参加                      | 0             | 无即时效果                             |
| 2. 备赛（准备） | 报名后的 1~2 回合内，每回合可选择「竞赛备赛」 | 2/回合        | 每回合「备赛积分」+1，学力 +3，压力 +3 |
| 3. 参赛（结算） | 备赛期结束后的指定回合自动触发                | 0（自动结算） | 根据获奖等级给予奖励                   |

#### 竞赛类型与获奖概率

| 竞赛名称            | 级别               | 解锁条件                  | 获奖概率公式                                      |
| ------------------- | ------------------ | ------------------------- | ------------------------------------------------- |

### 4.9 志愿者 / 青协活动系统

> 参加志愿服务是降低黑暗值的主要途径之一，同时提升领导力和社会资本。

- **解锁条件：** 回合一选择加入「青协」，或大一下学期起任何回合均可报名参加志愿活动
- **入口位置：** 「自由行动阶段」新增选项「参加志愿活动」

#### 志愿活动类型

### 4.10 补充事件系统

#### 4.10.1 班会发言 / 主持

- **触发时机：** 回合二（10 月班会）强制触发；此后每 2 回合概率触发（30%）
- **前提条件：** 玩家出席班会（未旷课）

| 选项               | 行动点消耗 | 效果                                                               |
| ------------------ | ---------- | ------------------------------------------------------------------ |
| 默默听讲（不发言） | 0          | 无正面效果                                                         |
| 主动发言           | 0          | 领导力 +2，魅力 +1，心情 +2；若学力 ≥ 150：额外 NPC 好感度总和 +5 |
| 主动申请主持班会   | 1          | 领导力 +2，魅力 +3，勇气 +2；若担任班委：额外领导力 +2             |

#### 4.10.2 组织社团活动

- **解锁条件：** 社团活动次数 ≥ 10 且 领导力 ≥ 60（相当于在社团中积累了足够经验和影响力）
- **入口位置：** 解锁后，「自由行动阶段」新增选项「组织社团活动」

| 活动规模                      | 行动点消耗 | 效果                                                                       | 成功率                         |
| ----------------------------- | ---------- | -------------------------------------------------------------------------- | ------------------------------ |
| 小型活动（聚餐/观影）         | 2          | 领导力 +2，社团活动次数 +1，NPC 好感度总和 +10，心情 +5                    | 80% + 领导力×0.1              |
| 中型活动（比赛/演出）         | 3          | 领导力 +4，社团活动次数 +2，魅力 +3，NPC 好感度总和 +15                    | 60% + 领导力×0.1              |
| 大型活动（校级晚会/跨校联谊） | 4          | 领导力 +6，社团活动次数 +3，魅力 +5，NPC 好感度总和 +25，金钱 -300（经费） | 40% + 领导力×0.1 + 魅力×0.05 |

**活动失败后果：** 心情 -5，压力 +5，领导力 -2

#### 4.10.3 校园跑代跑补充

> 在 §12.3 已提及代跑选项，此处补充属性影响。

- **代跑价格：** 5~8 元/次（越靠近期末越贵）
- **选择代跑时：**
  - 体魄不增加（没有自己跑）
  - 负罪感 +1/次
  - 黑暗值 +1/次（轻微的学术不诚信）
  - 被发现概率 5%/次：被发现后该次不计入校园跑次数，且压力 +5

### 4.11 兼职 / 打工系统

> 除了初始选项 G「勤工俭学」的固定收入外，玩家也可以主动选择各种兼职来赚取额外收入。

- **解锁时机：** 大一下学期（回合六）起可参加兼职
- **入口位置：** 「自由行动阶段」新增选项「兼职打工」

#### 正规兼职类型

| 兼职类型                   | 行动点消耗 | 金钱收益                       | 属性效果         | 解锁条件              |
| -------------------------- | ---------- | ------------------------------ | ---------------- | --------------------- |
| 食堂打工                   | 1          | +200/回合                      | 体魄 +1          | 无                    |
| 家教                       | 2          | +500~800/回合                  | 学力 +2，魅力 +1 | 学力 ≥ 120           |
| 快递分拣                   | 1          | +300/回合                      | 体魄 +2，心情 -2 | 无                    |
| 校园超市收银               | 1          | +250/回合                      | 魅力 +1          | 无                    |
| 线上兼职（翻译/设计/编程） | 2          | +600~1200/回合                 | 学力 +3          | 通过四级或学力 ≥ 150 |
| 自媒体运营                 | 2          | +300~2000/回合（随粉丝量增长） | 魅力 +3          | 魅力 ≥ 100           |

**兼职与学业冲突：**

- 每回合最多选择 1 种兼职
- 兼职回合，若同时选择了自习：压力 +3（疲劳）
- 连续 3 回合兼职且未自习：学力 -5，摆烂值 +3（学业懈怠）

**兼职计数：** 每完成 1 回合兼职，「兼职/打工次数」+1
