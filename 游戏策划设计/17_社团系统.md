
## 17. 社团系统

> Club & Organization System
> 完整技术细节请参阅 `specs/club-system-社团系统.md`。

### 17.1 系统总览与设计理念

社团是大学生活的核心组成部分，在游戏中承担三个设计目标：

1. **属性增长的差异化路径** — 不同社团提供不同属性的高效增长通道
2. **社交网络扩展器** — 结识NPC、积累好友和好感度的主要途径
3. **独特体验供给** — 每个社团自带专属小游戏和事件链

#### 组织分类

| 类别                           | 包含                                               | 名额                   | 性质                                 |
| ------------------------------ | -------------------------------------------------- | ---------------------- | ------------------------------------ |
| **社团/学生组织**（7个） | 跑协、篮球社、吉他社、辩论社、青协、学生会、动漫社 | 最多加入**2个**  | 兴趣驱动，自由加入退出               |
| **官方政治组织**（2个）  | 校团委、党建班                                     | **不占社团名额** | 政治路径，有门槛和考核，不可主动退出 |

> 校团委和党建班不占社团的2个名额，但会额外消耗行动点——同时加社团+官方组织，行动点会非常紧张。

### 17.3 加入与退出机制

#### 社团加入流程

**回合一「百团大战·社团招新」（强制事件）：** 展示7个社团卡片，玩家选择最多加入2个。

- 暂时不加入：不会立刻惩罚，但大二上仍未加入任何社团 → 「社交孤立警告」：心情-10，摆烂值+3
- 每学年第一回合有「社团补招」机会

**加入条件：**

| 社团   | 条件                       | 说明        |
| ------ | -------------------------- | ----------- |
| 跑协   | 无                         | 来者不拒    |
| 篮球社 | 体魄 ≥ 80                 | 需体能基础  |
| 吉他社 | 无                         | 零基础欢迎  |
| 辩论社 | 学力 ≥ 90                 | 需知识储备  |
| 青协   | 无                         | 只要有心    |
| 学生会 | 魅力 ≥ 90 或 领导力 ≥ 60 | 需面试通过  |
| 动漫社 | 无                         | 喜欢ACG即可 |

**退出机制：**

- 主动退出：2回合冷却期，担任职位则领导力-5（退出学生会额外-10）
- 被动退出：连续5回合未参加 → 自动被踢出，摆烂值+5，社团NPC好感度-20

#### 官方组织加入

**校团委（回合二面试）：** 需领导力≥60、学力≥100、负罪感≤40。成功率 = 40% + 领导力×0.15 + 学力×0.05 + 班委加成20%。

**党建班（回合十二「入党申请」）：** 需学力≥110、领导力≥55、负罪感≤30、GPA≥2.5。筛选通过后进入党建班学习。

**淘汰机制：** 校团委/党建班均有学期考核，不达标则被淘汰（不可主动退出）。党建班被淘汰后无法再次申请。

### 17.4 社团活动行动

| 项目       | 说明                              |
| ---------- | --------------------------------- |
| 行动点消耗 | **-2**                      |
| 每回合限制 | 每个社团最多参加1次               |
| 属性效果   | 见§17.2属性对比表                |
| 额外效果   | 社团活动次数+1，社团NPC好感度+3~5 |

### 17.5 社团事件链概览

每个社团都有贯穿四年的5阶段故事线（入门→融入→选择→挑战→高光），由社团活动次数和NPC好感度驱动推进。完整事件链详见 `specs/club-system-社团系统.md` §5。

关键节点：

- **活动次数≥10：** 触发社团选择事件（道德/路线分岔）
- **活动次数≥15：** 触发社团挑战事件（含小游戏）
- **活动次数≥20：** 触发社团高光事件（解锁对应结局条件）

### 17.6 职务竞选与晋升系统

> 所有校园职务的获取都需要通过竞选或考核，担任职务后行动点上限减少（详见 §6.4）。职务带来领导力、结算积分和高星结局条件，但压缩玩家的自由行动空间。

---

#### 17.6.1 班委竞选事件

**触发时机：** 回合二（大一上 10 月班会），强制弹出竞选界面
**竞选流程：** 玩家选择是否参选 → 选择竞选职位 → 发表竞选宣言 → 系统判定结果

**可竞选职位与成功率：**

| 职位                           | 竞争难度 | 成功率公式                                               | 行动点上限减少 | 当选效果                                  |
| ------------------------------ | -------- | -------------------------------------------------------- | -------------- | ----------------------------------------- |
| 普通班委（文艺/体育/生活委员） | 低       | 60% + 魅力×0.1 + NPC好感度总和×0.01                    | -1             | 领导力 +5，心情 +3                        |
| 学习委员                       | 中       | 45% + 学力×0.05 + 魅力×0.1 + GPA加成（GPA≥3.0则+15%） | -2             | 领导力 +8，学力 +3                        |
| 团支书                         | 中       | 45% + 领导力×0.1 + 魅力×0.1                            | -2             | 领导力 +10，解锁校团委推荐加成            |
| 班长                           | 高       | 30% + 领导力×0.1 + 魅力×0.15 + NPC好感度总和×0.02     | -3             | 领导力 +15，心情 +5，班委补贴 +200元/回合 |

**竞选宣言选项（影响成功率修正）：**

| 宣言风格                           | 效果                                                 |
| ---------------------------------- | ---------------------------------------------------- |
| 务实型（「我会认真做好每一件事」） | 成功率 +5%，当选后NPC好感度总和 +5                   |
| 幽默型（「选我，选我，选我」）     | 成功率 +10%（受欢迎），但若落选心情额外 -3（丢面子） |
| 慷慨型（「请大家吃饭！」）         | 成功率 +15%，但当选后立刻扣除 -200 元（请客费）      |
| 放弃发言                           | 成功率 -10%                                          |

**竞选结果：**

| 结果   | 效果                                                            |
| ------ | --------------------------------------------------------------- |
| 当选   | 对应职位效果生效；标记「担任班委」= 是；每回合自动获得领导力 +2 |
| 落选   | 心情 -3，无其他惩罚                                             |
| 未参选 | 无效果                                                          |

**任期与换届：** 班委任期为一学年（10 回合）。每学年第一回合自动触发换届，在任班委可选择「连任」（无需重新竞选，自动保留职位）或「卸任」（领导力 -3）。

#### 17.6.2 学生会晋升体系

> 学生会是领导力最强的成长路径，但晋升阶梯长、压力大，行动点消耗随职位攀升。

**晋升路径：** 干事 → 部长 → 副主席 → 主席

**加入（干事）：** 回合一社团招新时选择加入学生会，需魅力 ≥ 90 或领导力 ≥ 60，通过面试后成为干事。

| 职位   | 晋升触发时机                 | 晋升条件                                                       | 成功率公式                                      | 行动点上限减少 | 晋升效果                                  |
| ------ | ---------------------------- | -------------------------------------------------------------- | ----------------------------------------------- | -------------- | ----------------------------------------- |
| 干事   | 加入即是                     | —                                                             | —                                              | -2             | 领导力 +3/回合                            |
| 部长   | 回合十（大一下学期末考核）   | 活动次数 ≥ 8 + 领导力 ≥ 80 + 赵恒好感度 ≥ 40                | 40% + 领导力×0.15 + 魅力×0.1                  | -3             | 领导力 +5/回合，解锁「部门管理」事件      |
| 副主席 | 回合十五（大二上学期末换届） | 活动次数 ≥ 16 + 领导力 ≥ 120 + 赵恒好感度 ≥ 60              | 30% + 领导力×0.15 + 魅力×0.15                 | -4             | 领导力 +6/回合，综测 +3 分                |
| 主席   | 回合二十（大二下学期末换届） | 活动次数 ≥ 25 + 领导力 ≥ 160 + 赵恒好感度 ≥ 80 + GPA ≥ 2.5 | 20% + 领导力×0.15 + 魅力×0.15 + 赵恒好感×0.1 | -5             | 领导力 +8/回合，综测 +5 分，结算积分 +200 |

**晋升事件流程：**

触发时弹出竞选界面，包含以下阶段：

1. **述职报告**——系统读取玩家本学期活动数据，自动生成述职要点，玩家选择汇报风格：

| 汇报风格 | 效果                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| 如实汇报 | 成功率不变                                                                   |
| 夸大业绩 | 成功率 +10%，但黑暗值 +3，负罪感 +2（如被识破：成功率反 -20%，识破概率 15%） |
| 谦虚低调 | 成功率 -5%，但赵恒好感度 +5                                                  |

2. **投票环节**——系统判定成功率，宣布结果

**晋升失败后果：** 心情 -5，压力 +5，可在下一次换届时重新竞选
**降职/免职：** 连续 2 回合未参加学生会活动 → 自动降一级；摆烂值 ≥ 60 → 免职降为普通成员

#### 17.6.3 社团换届选举

> 适用于七大社团的社长/会长职位竞选。

**触发时机：** 回合十五（大二下学期初），所有已加入的社团同步触发换届

**晋升路径：** 普通成员 → 骨干/副社长 → 社长/会长

| 职位        | 晋升条件                             | 成功率公式                                        | 行动点上限减少 | 晋升效果                      |
| ----------- | ------------------------------------ | ------------------------------------------------- | -------------- | ----------------------------- |
| 普通成员    | 加入即是                             | —                                                | 0              | 属性加成 ×1.0                |
| 骨干/副社长 | 活动次数 ≥ 10 + 社团NPC好感度 ≥ 50 | 自动晋升（无需竞选）                              | -1             | 属性加成 ×1.3                |
| 社长/会长   | 活动次数 ≥ 18 + 社团NPC好感度 ≥ 70 | 50% + 领导力×0.1 + 魅力×0.1 + 社团NPC好感×0.15 | -2             | 属性加成 ×1.5，结算积分 +100 |

**换届选举事件流程：**

1. **系统检测**——判断玩家是否满足社长竞选条件（活动次数 + NPC好感度）
2. **竞选演说**——若满足条件，弹出选项：

| 选项                 | 效果                                       |
| -------------------- | ------------------------------------------ |
| 参加竞选             | 进入判定流程                               |
| 推荐他人（放弃竞选） | 社团NPC好感度 +10（谦让的人缘），领导力 -2 |
| 不参加也不推荐       | 无效果                                     |

3. **结果公布**——
   - 当选：心情 +8，领导力 +10，弹出叙事「社团成员们为你鼓掌，你正式成为新任社长」
   - 落选：心情 -3，社团NPC好感度 +3（「虽然没选上，但大家看到了你的努力」）

**社长任期：** 回合十五至毕业。社长可在任意回合选择「卸任」，卸任后降为骨干/副社长，领导力 -5。

#### 17.6.4 校团委晋升体系

> 校团委是仕途线最核心的组织，晋升难度最高，但收益也最大。

**晋升路径：** 干事 → 副书记

| 职位   | 晋升触发时机                 | 晋升条件                                                                        | 成功率公式                           | 行动点上限减少 | 晋升效果                                  |
| ------ | ---------------------------- | ------------------------------------------------------------------------------- | ------------------------------------ | -------------- | ----------------------------------------- |
| 干事   | 回合二面试通过即是           | —                                                                              | —                                   | -2             | 领导力 +4/回合                            |
| 副书记 | 回合二十（大二下学期末考核） | 活动次数 ≥ 20 + 领导力 ≥ 150 + 何正清好感度 ≥ 70 + GPA ≥ 3.0 + 负罪感 ≤ 30 | 25% + 领导力×0.15 + 何正清好感×0.2 | -4             | 领导力 +8/回合，综测 +5 分，结算积分 +250 |

**晋升事件「组织考察」：**

> 校团委晋升不是公开竞选，而是由何正清（副书记NPC）进行组织考察。

1. **考察谈话**——何正清主动找玩家谈话，弹出对话事件：

> *「坐。最近组织上一直在关注你。」何正清翻开一个笔记本，上面密密麻麻记着你这一年的表现。「说说你对自己这段时间工作的看法。」*

| 回答选项                               | 效果                                      |
| -------------------------------------- | ----------------------------------------- |
| 实事求是总结                           | 成功率不变，何正清好感 +5（「态度端正」） |
| 主动揽责+表决心                        | 成功率 +15%（「有担当」）                 |
| 推功揽过（夸大团队功劳，隐瞒个人失误） | 成功率 +5%，黑暗值 +3                     |
| 质疑组织安排                           | 成功率 -20%，何正清好感 -10，但勇气 +5    |

2. **背景审查**——系统自动检查玩家数据：

   - 黑暗值 ≥ 40：晋升直接失败（「组织上发现一些问题……」）
   - 旷课次数 ≥ 10：成功率 -15%
   - 入党流程进行中：成功率 +10%
3. **结果通知**——

   - 晋升成功：领导力 +15，心情 +10，压力 +10（「更大的责任意味着更大的压力」）
   - 晋升失败：心情 -5，但保留干事身份

**免职条件：** 负罪感 ≥ 60 或摆烂值 ≥ 50 或连续 3 回合未参加校团委活动 → 自动免职降为无职务状态，且不可重新加入

### 17.7 社团小游戏

每个社团/组织对应1个专属小游戏：

| 小游戏        | 所属   | 玩法类型                                | 优先级 |
| ------------- | ------ | --------------------------------------- | ------ |
| 🏃 晨跑打卡   | 跑协   | 节奏反应类（躲避障碍）                  | P1     |
| 🏀 投篮挑战   | 篮球社 | 精准时机类（蓄力条）                    | P0     |
| 🎸 节奏弹奏   | 吉他社 | 音乐节奏类（简化音游）                  | P0     |
| 🗣️ 辩论对决 | 辩论社 | 策略选择+快速反应（石头剪刀布+属性）    | P1     |
| 📋 志愿任务   | 青协   | 资源分配+时间管理                       | P2     |
| 🏛️ 会议决策 | 学生会 | 多选博弈+后果推演（预算分配）           | P2     |
| 🎨 动漫创作   | 动漫社 | 绘画/Cosplay搭配类（限时拼图+审美选择） | P1     |
| 📊 团务统筹   | 校团委 | 多任务调度+优先级排序                   | P2     |
| 📝 思想汇报   | 党建班 | 叙事选择+言行一致性检测                 | P1     |

> 思想汇报是独特小游戏——系统会读取玩家实际游戏数据（志愿次数、旷课记录、消费记录等）来验证回答的真实性。

### 17.8 社团与其他系统的联动

| 系统     | 联动方式                                                    |
| -------- | ----------------------------------------------------------- |
| 结局系统 | 社团活动次数≥20 → 解锁对应结局；入党 → 仕途线高位结局    |
| 属性系统 | 青协负罪感衰减-1/回合；党建班负罪感衰减-2/回合              |
| 校园跑   | 加入体育社团 → 校园跑目标32次（否则52次）                  |
| GPA系统  | 辩论社口试+5%；学生会综测+2分；校团委综测+3分；党员综测+2分 |
| 金钱系统 | 各社团活动花费不同（跑协-20/次、吉他社首次-300买吉他）      |
| NPC系统  | 9个社团/组织NPC（含何正清、宋明德），加入后解锁交互         |

### 17.9 入党全流程时间线

```
回合12 (大二上10月)     提交入党申请书          → partyStatus = Applied
  ↓ 筛选通过
回合13~14               进入党建班学习          → partyStatus = Activist
  ↓ 持续参加
回合16~20               积累党建班活动次数
  ↓ 活动次数≥15 + 负罪感≤30
回合20左右              预备党员宣誓            → partyStatus = Probationary
  ↓ 4回合考察期
回合24左右 (大三上末)    转为正式党员            → partyStatus = FullMember
  ↓ 毕业
结局判定                 入党 = true → 仕途线高位结局解锁
```

> **总耗时约 12 回合（大二上→大三上），占总游戏的 30%。** 玩家必须在大二上就做出「要不要走仕途线」的战略决策。
