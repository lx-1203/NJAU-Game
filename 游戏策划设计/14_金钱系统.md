## 14. 金钱系统

> Money System

### 14.1 基础经济数值

| 项目                                                      | 数值                                                                          |
| --------------------------------------------------------- | ----------------------------------------------------------------------------- |
| 初始金钱                                                  | 8000 元                                                                       |
| 每月生活费（每回合）                                      | 1500 元                                                                       |

### 14.2 收入来源汇总

> **设计理念：** 收入来源有限且大多附带条件或行动点代价，迫使玩家在「赚钱」和「提升属性」之间做取舍，符合现实中大学生的经济状况。

#### 14.2.1 固定收入

| 收入来源               | 金额     | 频率       | 条件           |
| ---------------------- | -------- | ---------- | -------------- |
| 每月生活费（父母打钱） | +1500 元 | 每回合自动 | 无条件         |
| 勤工俭学加成           | +500 元  | 每回合自动 | 开局选择选项 G |
| 班委补贴               | +200 元  | 每回合自动 | 班委竞选成功后 |

#### 14.2.2 主动收入（兼职系统）

> 兼职是玩家主动赚钱的核心途径，但每种兼职都需要消耗行动点，部分还有属性门槛。**每回合最多选择 1 种兼职。**
>
> **兼职类型详情及效果**：[详见 §4.11 兼职/打工系统](04_成就事件与结局.md#411-兼职--打工系统)
>
> **收入汇总（金钱视角）：**

| 兼职类型     | 收入范围          | 行动点消耗 | 门槛条件              | 金钱效率(元/AP) |
| ------------ | ----------------- | ---------- | --------------------- | --------------- |
| 食堂打工     | +200 元/回合      | 1          | 无                    | 200             |
| 家教         | +500~800 元/回合  | 2          | 学力 ≥ 120           | 250~400         |
| 快递分拣     | +300 元/回合      | 1          | 无                    | 300             |
| 校园超市收银 | +250 元/回合      | 1          | 无                    | 250             |
| 线上兼职     | +600~1200 元/回合 | 2          | 通过四级或学力 ≥ 150 | 300~600         |
| 自媒体运营   | +300~2000 元/回合 | 2          | 魅力 ≥ 100           | 150~1000        |

#### 14.2.3 被动/一次性收入

| 收入来源            | 金额          | 触发条件                                |
| ------------------- | ------------- | --------------------------------------- |
| 奖学金（三等）      | +1000 元      | 学期 GPA ≥ 3.0                         |
| 奖学金（二等）      | +2000 元      | 学期 GPA ≥ 3.5                         |
| 奖学金（一等/国奖） | +5000 元      | 学期 GPA ≥ 3.8 且竞赛获奖 ≥ 1         |
| 竞赛奖金            | +500~2000 元  | 竞赛获奖时一次性                        |
| 助学金              | +2000 元/学期 | 初始选择「家境困难」背景（与选项G互斥） |
| 卖二手教材          | +100~300 元   | 每学期末可选，出售上学期课本            |
| NPC 红包/转账       | +50~200 元    | 好友/恋人 NPC 在特定节日触发            |
| 捡到钱              | +10~50 元     | 低概率随机事件，受幸运影响              |

### 14.3 消费项目汇总

> **设计理念：** 消费分为「刚需消费」（无法避免或避免代价很大）和「可选消费」（花钱换属性/心情/好感度的投资决策）。每笔花销都有现实依据，让玩家感受到真实的大学生活经济压力。
>
> **核心体验目标：** 正常玩家基本月光，想攒钱就得省着花或去打工；想过得舒服就得接受月光。

#### 14.3.1 大额刚性支出（每学期强制）

| 项目        | 金额          | 频率         | 备注                                               |
| ----------- | ------------- | ------------ | -------------------------------------------------- |
| 学费        | -5000 元      | 每学期回合一 | 强制扣除，不可避免                                 |
| 学杂费/书费 | -960 元（约） | 每学期回合一 | 按科目累计；触发「神秘二手书学长」并购买可减免部分 |

#### 14.3.2 饮食消费（每回合）

> 详见 §4.1 固定事件「吃饭」，此处汇总花费。饮食是每回合必选项（不吃饭心情 -5、行动点 -1）。

| 饮食档次           | 花费/回合 | 效果概述               |
| ------------------ | --------- | ---------------------- |
| 泡面（最低档）     | -3 元     | 行动点 +1，但负罪感 +2 |
| 食堂套餐（标准档） | -12 元    | 行动点 +1，心情 +1     |
| 黄教授烧饼         | -8 元     | 心情 +3                |
| 南农烧鸡           | -15 元    | 行动点 +2              |
| 外卖（高档）       | -30 元    | 心情 +5，行动点 +1     |

**每回合可选加购（叠加在正餐之上）：**

| 加购项    | 花费      | 效果             | 说明                 |
| --------- | --------- | ---------------- | -------------------- |
| 奶茶      | -15 元    | 心情 +2          | 续命饮品，每回合可选 |
| 咖啡      | -20 元    | 行动点 +1        | 适合自习前购买       |
| 水果/零食 | -30~80 元 | 心情 +2，体魄 +1 | 每回合可选           |

#### 14.3.3 日常生活消费（周期性刚需）

> 这些消费不大但持续存在，不处理会产生负面效果。模拟现实中大学生「看不见的开销」。

| 项目                               | 花费       | 频率                  | 不支付的后果           |
| ---------------------------------- | ---------- | --------------------- | ---------------------- |
| 日用品补给（洗衣液/纸巾/沐浴露等） | -50~100 元 | 每 2 回合自动提醒     | 不买则心情 -3/回合     |
| 宿舍电费                           | -30~80 元  | 每回合自动扣除        | 不可避免               |
| 话费/流量                          | -30~50 元  | 每回合自动扣除        | 不可避免               |
| 打印/复印费                        | -20~50 元  | 期中/期末回合自动触发 | 不打印则考试通过率 -5% |

#### 14.3.4 穿着/形象消费

> 形象消费是提升魅力值的重要途径，也是恋爱线和社交线的隐性投资。

| 项目               | 花费        | 频率/条件      | 效果                         |
| ------------------ | ----------- | -------------- | ---------------------------- |
| 换季置装（买衣服） | -300~800 元 | 每学期开始可选 | 魅力 +3~8（按花费档次）      |
| 理发               | -30~80 元   | 每 2 回合可选  | 魅力 +2~3                    |
| 染发/烫发          | -150~300 元 | 自由选择       | 魅力 +4~5                    |
| 买鞋               | -200~800 元 | 自由选择       | 魅力 +3~6，校园跑体魄加成 +1 |
| 护肤品/化妆品      | -100~500 元 | 每学期可选     | 魅力 +2~5（持续整学期）      |

#### 14.3.5 娱乐/社交消费

> 娱乐消费是恢复心情、维系人际关系的核心手段。消费越高，社交效果越好——模拟现实中「花钱维系社交」的规律。
>
> **出校门子活动详情**：[详见 §4.1 固定事件「出校门」子活动](04_成就事件与结局.md#41-固定事件每个回合固定触发)

#### 14.3.6 学习/考证投入

> 花钱提升学业成功率，是「金钱→属性」转化的关键消费通道。

| 项目             | 花费          | 频率/条件            | 效果                      |
| ---------------- | ------------- | -------------------- | ------------------------- |
| 四六级资料/网课  | -100~300 元   | 报名四六级后可选     | 四六级通过率 +10%         |
| 考研资料/辅导班  | -500~3000 元  | 大三后，选择考研路线 | 考研成功率大幅提升        |
| 考公教材/网课    | -300~2000 元  | 大三后，选择考公路线 | 考公成功率大幅提升        |
| 考驾照           | -3500~5000 元 | 大二后解锁，一次性   | 魅力 +5，部分职场结局加成 |
| 竞赛报名费       | -50~200 元    | 竞赛事件触发         | 获奖概率受学力影响        |
| 计算机考试报名费 | -100 元       | 指定回合触发         | 一次性                    |

#### 14.3.7 人情/社交消费

> 人际关系需要金钱维护——现实中大学生的「隐性社交成本」。好感度越高、关系越深，花钱的场合越多。

| 项目                    | 花费          | 触发条件                           | 效果                             |
| ----------------------- | ------------- | ---------------------------------- | -------------------------------- |
| NPC 一起吃饭            | -10 元/次     | 好感 ≥ 20                         | 好感 +10，心情 +3                |
| NPC 送礼物              | -50~200 元/次 | 好感 ≥ 40                         | 好感 +20                         |
| NPC 生日礼物            | -100~500 元   | NPC 生日事件触发                   | 好感 +10~15；不送则好感 -10      |
| 随份子（室友/同学请客） | -100~300 元   | 随机事件触发（**不可拒绝**） | 好感 +5；拒绝则好感 -10，魅力 -3 |
| 社团会费                | -50~200 元    | 加入社团时一次性                   | 解锁社团活动线                   |
| 班级活动 AA             | -30~80 元     | 班会/班级活动触发                  | 不参加则领导力 -2                |

#### 14.3.8 恋爱消费

> 恋爱是游戏中**最大的持续性消费坑**——进入恋爱状态后，每回合都有维护成本。这符合现实中「恋爱烧钱」的体验。

| 项目                      | 花费           | 触发条件               | 效果                               |
| ------------------------- | -------------- | ---------------------- | ---------------------------------- |
| 表白花束/礼物             | -100~500 元    | 选择告白时             | 告白成功率 +15~30%                 |
| 约会开销（餐厅/娱乐）     | -100~300 元/次 | 恋爱后每 2 回合可选    | 恋人好感度不衰减                   |
| 纪念日礼物                | -200~800 元    | 恋爱后特定回合自动触发 | 好感 +10；**忘记则好感 -15** |
| 节日礼物（情人节/圣诞等） | -100~500 元    | 节日回合触发           | 好感 +5~8；不送则好感 -5           |
| 异地车票（寒暑假见面）    | -200~600 元    | 恋人异地时寒暑假可选   | 维持好感度，不去则好感 -10/学期    |

> **恋爱经济压力测算：** 维持恋爱状态平均每回合额外消费约 100~250 元，每学期约 500~1250 元。这意味着恋爱中的玩家几乎必然月光，不打工很难同时维持恋爱和其他消费。

#### 14.3.11 健康/医疗消费（被动触发）

> 不可控消费，模拟现实生活中的意外开销，增加经济系统的不确定性。

| --------------------------- | --------------- | ------------------------- | -------------------- |

### 14.4 收支平衡分析

> 以下为不同玩法风格下的单学期（5 回合）经济测算，帮助策划验证数值平衡。

#### 场景 A：极度节省玩家（不社交、不娱乐、只吃泡面）

| 项目                   | 金额                |
| ---------------------- | ------------------- |
| 收入：生活费 ×5       | +7,500              |
| 支出：学费             | -5,000              |
| 支出：书费             | -960                |
| 支出：饮食（泡面 ×5） | -15                 |
| 支出：电费+话费 ×5    | -300                |
| **净余额**       | **+1,225 元** |

> 能攒钱，但心情和负罪感会快速恶化，走向抑郁-治疗的消耗循环。

#### 场景 B：正常生活玩家（适度消费、偶尔社交）

| 项目                                 | 金额                             |
| ------------------------------------ | -------------------------------- |
| 收入：生活费 ×5                     | +7,500                           |
| 支出：学费 + 书费                    | -5,960                           |
| 支出：饮食（食堂 + 偶尔外卖 + 奶茶） | -200                             |
| 支出：日用品 + 电费 + 话费           | -350                             |
| 支出：理发 ×2                       | -100                             |
| 支出：出校门（购物/大餐等）×1       | -400                             |
| 支出：看电影 ×1 + 聚餐 ×1          | -200                             |
| 支出：打印费                         | -40                              |
| 支出：零食/水果                      | -150                             |
| 支出：NPC 社交（送礼/聚餐 AA）       | -200                             |
| **净余额**                     | **≈ -100 元（基本月光）** |

> **这是设计目标：正常玩家基本月光**，和现实大学生体验一致。

#### 场景 C：恋爱中的玩家

| 项目                  | 金额                         |
| --------------------- | ---------------------------- |
| 收入：生活费 ×5      | +7,500                       |
| 支出：场景 B 全部支出 | -7,600                       |
| 支出：约会开销 ×2    | -400                         |
| 支出：纪念日/节日礼物 | -300                         |
| **净余额**      | **≈ -800 元（赤字）** |

> 恋爱中的玩家必须打工或削减其他消费，否则会滑向透支。

#### 场景 D：高消费 + 心理危机玩家

| 项目                    | 金额                                     |
| ----------------------- | ---------------------------------------- |
| 收入：生活费 ×5        | +7,500                                   |
| 支出：场景 B 全部支出   | -7,600                                   |
| 支出：代跑 ×32         | -200                                     |
| 支出：抑郁治疗 ×3 回合 | -1,500~3,000                             |
| 支出：冲动消费 ×3 回合 | -150~600                                 |
| **净余额**        | **-1,950 ~ -3,900 元（严重赤字）** |

> 进入死亡螺旋：没钱 → 只能吃泡面 → 负罪感↑ → 冲动消费 → 更没钱 → 网贷 → 破产结局。

### 14.5 负债与危机机制

| 金钱状态        | 触发效果                                                        |
| --------------- | --------------------------------------------------------------- |
| 金钱 < 200 元   | 触发「快揭不开锅了」提示，吃饭只能选泡面，娱乐/社交消费全部锁定 |
| 金钱 < 0 元     | 进入「透支」状态，每回合额外压力 +5，不可进行任何可选消费       |
| 金钱 < -2000 元 | 触发「网贷」前置事件，玩家选择是否借网贷                        |
| 金钱 < -5000 元 | 直接触发「网贷破产」坏结局                                      |

### 14.6 经济系统设计总结

| 设计要点                    | 说明                                                                                                                                                  |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| **收入有限**          | 无条件固定收入仅有生活费（1500/回合），兼职赚钱需消耗宝贵的行动点                                                                                     |
| **消费分层**          | 刚需（电费/话费/饮食）→ 形象投资（衣服/理发）→ 社交维护（聚餐/礼物）→ 学业投入（资料/考证）→ 意外事件（生病/碎屏）→ 大额投资（驾照/考研班/手机） |
| **花钱有回报**        | 每笔消费都影响属性/好感/概率，玩家在做投资决策而非被白白扣钱                                                                                          |
| **社交绑定消费**      | 好感度越高 → 解锁的消费项越多（聚餐、约会、礼物），恋爱线是最大的「钱坑」                                                                            |
| **经济健康=心理健康** | 负罪感/压力系统通过「抑郁治疗费」和「冲动消费」与金钱联动，心态崩=钱也崩                                                                              |
| **死亡螺旋**          | 没钱→不社交/吃泡面→心情↓→抑郁→治疗费→更没钱→网贷→破产                                                                                         |
| **勤工俭学价值极高**  | 额外 500/回合 = 2500/学期，几乎是正常玩家净余额的全部安全边际                                                                                         |
