	
## 6. 角色属性设计

> Stats Design

### 6.1 核心属性

#### 学力

- **初始值：** 100 | **上限：** 无 | **下限：** 无
- **影响：** 期末周挂科概率、绩点属性
- **期末周挂科概率** = 原 50%，受特殊事件"期末周复习"影响，如果复习则增加通过率

#### 魅力

- **初始值：** 100 | **上限：** 无 | **下限：** 无
- **影响：** NPC 好感度的增加比例

**好感度计算公式：**

```
NPC好感度增加值 = 事件基础好感度增加值 × (1 + 魅力值 / 100)
```

> **举例：** 当前魅力值 60，好感事件基础增加值 +10 → 最终增加 = 10 × (1 + 0.6) = **+16**

#### 体魄

- **初始值：** 100 | **上限：** 无 | **下限：** 0
- **影响：** 2000 米/800 米（体测事件）、期末体育成绩、室内体测

#### 压力

- **初始值：** 0 | **上限：** 100 | **下限：** 0
- **本质：** 百分比计算
- **影响：** 心情、成功率、心理健康值

**心情减少值计算公式：**

```
心情减少值 = 坏事件基础扣除值 × (100% + 压力%)
```

> **举例：** 压力值 40，坏事件原数值 -10 → 实际扣除 = -10 × 1.4 = **-14**

**做事成功率计算公式：**

```
成功率 = 原成功率 - 压力值 × 0.1 + 幸运值 × 0.1 - 负罪感值 × 0.08
```

> **举例：** 压力值 40，原成功率 90%，幸运值 50 → 成功率 = 90 - 40×0.1 + 50×0.1 = **91%**

**抑郁触发：** 压力长期 > 60% 会持续降低心理健康值（-3/回合），间接增加抑郁风险。详见 [事件设计](#43-条件事件满足条件触发)

#### 心情

- **初始值：** 50 | **上限：** 100 | **下限：** 0
- **影响：** 心理健康值（心情长期 < 40% 会持续降低心理健康值 -3/回合，间接增加抑郁风险）

#### 领导力

- **初始值：** 50 | **上限：** 无 | **下限：** 0
- **影响：** 仕途结局判定、班委竞选成功率、学生组织晋升
- **定义：** 玩家在集体事务中展现的组织力和影响力，是仕途线（考公/考编）的核心驱动属性

**获取与消耗：**

| 行为                          | 领导力变化 |
| ----------------------------- | ---------- |
| 担任班委（每回合）            | +2         |
| 参加学生会/青协活动           | +3         |
| 组织社团活动                  | +2         |
| 入党积极分子学习              | +1         |
| 帮助 NPC                      | +1         |
| 班会发言/主持                 | +2         |
| 连续 3 回合不参加任何集体活动 | -5         |

#### 负罪感

- **初始值：** 0 | **上限：** 无 | **下限：** 0
- **影响：** 正向行动收益、行动点消耗、全局事件成功率、最终结局判定
- **定义：** 玩家主动选择与自我预期偏差带来的长期内耗，单纯娱乐、休息行为无法抵消其负面效果
- **机制：** 按数值区间划分梯度效果，区间效果**不叠加**，仅生效当前所处区间的最高阶惩罚

---

**轻度负罪感（30 - 59 点）：**

| 效果类型         | 描述                                                               |
| ---------------- | ------------------------------------------------------------------ |
| 正向行动收益打折 | 上课、自习获得的学力收益、操场锻炼获得的体魄收益均打**8 折** |
| 正向情绪恢复衰减 | 吃饭、睡觉、娱乐带来的心情恢复效果打**8 折**                 |
| 回合固定惩罚     | 每回合结束时，若仍处于该区间，自动**+3 压力、-2 心情**             |

---

**中度负罪感（60 - 89 点）：**

| 效果类型             | 描述                                                                                                                                                       |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 核心行动收益大幅衰减 | 上课、自习学力收益打**5 折**；体魄、勇气、运气相关正向行动打 **6 折**；所有主线目标（保研/考研/考公/创业/进企业）相关行动收益打 **5 折** |
| 行动点额外消耗       | 所有正向行动（上课、自习、复习、竞赛准备、实习申请、创业推进等）额外消耗**1 点行动点**                                                               |
| 社交效果衰减         | 魅力值带来的 NPC 好感度加成效果**失效 50%**；与 NPC 互动的心情收益打 **5 折**                                                                  |
| 全局成功率衰减       | `成功率 = 原成功率 - 压力值×0.1 + 幸运值×0.1 - 负罪感值×0.08`                                                                                         |
| 触发中度焦虑         | 触发**中度焦虑** debuff（详见 §4.3 焦虑系统）；**心理健康值 -2/回合**                                                                         |
| 回合固定惩罚         | 每回合结束自动**+6 压力、-5 心情**                                                                                                                         |

> **社交举例：** 魅力值 100，负罪感处于该区间，好感事件基础增加值 +10 → 最终好感度从 +20 变为 **+15**

> **成功率举例：** 负罪感 70，原成功率 90%，压力 40，幸运 50 → 成功率 = 90 - 40×0.1 + 50×0.1 - 70×0.08 = **85.4%**

---

**重度负罪感（90 - 149 点）：**

| 效果类型         | 描述                                                                                                                                                                                |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 正向行动权限锁定 | 高难度主线目标相关行动（竞赛报名、保研夏令营、创业项目启动、考公/考研系统复习等）**直接锁定无法选择**；剩余可执行正向行动收益打 **2 折**，额外消耗 **2 点行动点** |
| 核心属性持续衰减 | 每回合结束，学力、魅力、勇气中**随机 1 项永久 -5**                                                                                                                            |
| 触发重度焦虑     | 触发**重度焦虑** debuff（详见 §4.3 焦虑系统）；**心理健康值 -4/回合**；触发强制事件「夜里辗转反侧」                                                                    |
| 主线剧情锁死     | 人生规划事件中选定的核心目标相关主线事件**全部无法触发**                                                                                                                      |
| 金钱系统联动惩罚 | 每回合结束触发"冲动消费"，随机扣除**50-200 元**，无法规避；连续 3 回合处于该区间，**直接触发网贷相关坏结局前置事件**                                                    |
| 回合固定惩罚     | 每回合结束自动**+10 压力、-8 心情**                                                                                                                                                 |

---

**临界破防区间（≥ 150 点）：**

| 效果类型           | 描述                                                                                                          |
| ------------------ | ------------------------------------------------------------------------------------------------------------- |
| 全正向行动永久锁定 | 仅可选择睡觉、娱乐等消极选项，**无法执行任何正向行动**                                                  |
| 极速心理崩溃       | 每回合结束自动**+15 压力、-10 心情、心理健康值 -15**，1-2 回合内必然触发重度抑郁 →**"一跃解千愁"结局** |
| 系统联动恶化       | 每回合自动**摆烂值 +8**（极度内耗下迅速滑向退学深渊），同时 **黑暗值 +3**                         |

### 6.2 负面属性

#### 摆烂值

- **初始值：** 0 | **上限：** 无 | **下限：** 0
- **影响：** 退学结局判定、学业警告触发
- **定义：** 玩家持续选择消极、逃避行为后累积的惰性惯性。与学力相反——学力代表「我在进步」，摆烂值代表「我已放弃」

**增加摆烂值的行为：**

| 行为                           | 摆烂值变化 |
| ------------------------------ | ---------- |
| 旷课                           | +3/次      |
| 连续 2 回合不自习              | +5         |
| 连续 3 回合不参加社交/社团     | +4         |
| 睡觉超额（同回合睡觉 ≥ 2 次） | +2         |
| 连续 2 回合不完成校园跑        | +3         |

**降低摆烂值的行为：**

| 行为                        | 摆烂值变化 |
| --------------------------- | ---------- |
| 上课                        | -1/次      |
| 自习                        | -2/次      |
| 获得好成绩（期末 ≥ 80 分） | -5         |
| 完成校园跑学期目标          | -3         |
| 竞赛/四六级通过             | -5         |

**摆烂值梯度效果：** 区间效果**不叠加**，仅生效当前所处区间。

---

**轻度摆烂（30 - 59 点）：**

| 效果类型     | 描述                                     |
| ------------ | ---------------------------------------- |
| 学力增长衰减 | 上课、自习获得的学力收益打**8 折** |
| 摆烂快感     | 每回合心情**+2**（摆着爽），但压力不减   |
| 回合提示     | 系统提示「你最近是不是有点松懈了？」     |

---

**中度摆烂（60 - 89 点）：**

| 效果类型         | 描述                                 |
| ---------------- | ------------------------------------ |
| 学力增长大幅衰减 | 上课、自习学力收益打**5 折**   |
| 道德麻木         | 旷课不再增加负罪感（已经麻木了）     |
| 社交退缩         | NPC 互动获得的好感度打**7 折** |
| 回合固定效果     | 每回合自动**学力 -3**          |

---

**重度摆烂（90 - 119 点）：**

| 效果类型     | 描述                                               |
| ------------ | -------------------------------------------------- |
| 学业警告事件 | 触发「辅导员谈话」强制事件，玩家必须选择是否振作   |
| 正向行动抗拒 | 自习、背单词等正向行动额外消耗**1 点行动点** |
| 核心属性衰减 | 每回合自动**学力 -5、领导力 -3**             |

---

**彻底摆烂（≥ 120 点）：**

| 效果类型 | 描述                                       |
| -------- | ------------------------------------------ |
| 强制退学 | 直接触发**「半途而废」退学结局**，游戏结束 |

---

#### 黑暗值

- **初始值：** 0 | **上限：** 无 | **下限：** 0
- **影响：** 黑暗结局判定
- **定义：** 玩家主动选择灰色/违法行为后累积的道德滑坡值。能力越高 + 黑暗值越高 = 越高星的黑暗结局；能力本身无罪，用错了地方就是另一个故事

**增加黑暗值的行为：**

| 行为                         | 黑暗值变化 |
| ---------------------------- | ---------- |
| 考试作弊                     | +10        |
| 帮人代写/代考                | +8         |
| 参与灰色兼职（如诈骗群运营） | +15        |
| 欺骗 NPC 获取好处            | +5         |
| 同时与多个 NPC 恋爱（劈腿）  | +8         |
| 偷窃/造假                    | +12        |
| 倒卖/信息中介牟利            | +6         |

**降低黑暗值的行为：**

| 行为                               | 黑暗值变化 |
| ---------------------------------- | ---------- |
| 帮助 NPC（无偿）                   | -2         |
| 参加志愿者/青协活动                | -3         |
| 主动向受害者道歉（触发特殊选项时） | -5         |

> **设计意图：** 黑暗值不产生 debuff，它只是一个「方向标」——当黑暗值足够高时，结局判定进入黑暗线。与负罪感的区别在于：负罪感是内心的自我惩罚（影响效率），黑暗值是客观的行为记录（影响结局方向）。

#### 心理健康值（隐藏属性）

- **初始值：** 100 | **上限：** 100 | **下限：** 0
- **影响：** 抑郁系统触发、结局判定
- **定义：** 反映玩家角色整体心理状态的隐藏数值。玩家**看不到具体数字**，只能通过叙事文本、UI 变化等「症状」来感知。心理健康值受多种生活因素综合影响，不被单一行为直接击穿。
- **机制：** 按数值区间触发不同阶段的心理状态，每个阶段都有**自救窗口**（详见 §4.3 抑郁系统）

**降低心理健康值的因素（多维度）：**

| 因素                       | 心理健康值变化 | 说明         |
| -------------------------- | -------------- | ------------ |
| 压力长期 > 60%             | -3/回合        | 持续高压     |
| 心情长期 < 40%             | -3/回合        | 持续低落     |
| 连续 3 回合无社交互动      | -5             | 社交孤立     |
| 挂科                       | -8/次          | 学业挫败     |
| 被父母责骂（主线事件）     | -10            | 家庭压力     |
| 失恋/被拒绝                | -12            | 情感创伤     |
| 焦虑 debuff 生效中（中度） | -2/回合        | 焦虑侵蚀     |
| 焦虑 debuff 生效中（重度） | -4/回合        | 焦虑严重侵蚀 |
| 金钱 < 0（负债）           | -5/回合        | 经济焦虑     |
| 被校园霸凌/排挤事件        | -15            | 恶性社交事件 |
| 好友 NPC 好感度全部 < 30   | -4/回合        | 无人可倾诉   |

**提升心理健康值的因素：**

| 因素                        | 心理健康值变化 | 说明                                |
| --------------------------- | -------------- | ----------------------------------- |
| 与好友互动（好感 > 50）     | +3/次          | 社交支持                            |
| 出校门活动（爬山/看电影等） | +2/次          | 放松减压                            |
| 心理咨询（主动选择）        | +10~15/次      | 专业帮助（详见 §4.3 心理咨询系统） |
| 获得好成绩（期末 ≥ 80 分） | +5             | 自我肯定                            |
| 恋爱状态稳定                | +2/回合        | 情感支撑                            |
| 社团活动                    | +2/次          | 归属感                              |
| 运动（校园跑/体育）         | +2/次          | 生理调节                            |
| 给家人打电话                | +5             | 家庭温暖                            |
| 好友倾诉                    | +5             | 倾诉释放                            |

> **设计意图：** 心理健康值是一个「蓄水池」——日常中有很多细小的因素在悄悄消耗或补充它。只有当多种负面因素长期叠加、且玩家持续忽视自救手段时，才会最终滑向抑郁。这比「压力 > 70% 持续 3 回合就抑郁」真实得多。

### 6.3 稀有属性

| 属性           | 初始值 | 上限 | 下限 | 影响                       |
| -------------- | ------ | ---- | ---- | -------------------------- |
| **勇气** | 50     | 100  | 0    | 影响主线事件"创业"的成功率 |
| **幸运** | 50     | 100  | 0    | 影响主线事件的成功率       |

**幸运的成功率公式：**

```
成功率 = 原成功率 - 压力值 × 0.1 + 幸运值 × 0.1 - 负罪感值 × 0.08
```

### 6.4 消耗数值

| 数值             | 类型   | 说明                                                                                                                              |
| ---------------- | ------ | --------------------------------------------------------------------------------------------------------------------------------- |
| **金钱**   | 消耗品 | 购买东西、月底天天吃泡面、负债过多可能进入"网贷破产"坏结局                                                                        |
| **行动点** | 消耗品 | 每回合基础上限 20 点，担任职务会减少上限（详见 §6.4 职务行动点消耗）；回合结束自动补充至上限，可通过伙食/好事件/其他方式额外补充 |

### 6.5 职务行动点消耗

> 担任校园职务会**永久减少行动点上限**（在任期间），模拟处理公务、开会、协调事务占用的精力。职务越大、成就越高，占用越多。多个职务的扣减**叠加计算**。

**班委职务：**

| 职务                                     | 行动点上限减少 | 说明                       |
| ---------------------------------------- | -------------- | -------------------------- |
| 普通班委（文艺委员/体育委员/生活委员等） | -1             | 事务较轻，偶尔组织活动     |
| 学习委员                                 | -2             | 需要收发作业、协调老师     |
| 团支书                                   | -2             | 需要组织团日活动、传达通知 |
| 班长                                     | -3             | 班级核心管理者，事务最多   |

**社团职务：**

| 职务            | 行动点上限减少 | 说明                     |
| --------------- | -------------- | ------------------------ |
| 社团普通成员    | 0              | 参加活动即可，无管理负担 |
| 社团骨干/副社长 | -1             | 协助管理，分担社长工作   |
| 社长/会长       | -2             | 全面负责社团运营         |

**学生会职务：**

| 职务         | 行动点上限减少 | 说明                 |
| ------------ | -------------- | -------------------- |
| 学生会干事   | -2             | 执行层，跑腿+会议多  |
| 学生会部长   | -3             | 管理一个部门         |
| 学生会副主席 | -4             | 分管多个部门         |
| 学生会主席   | -5             | 最高职务，事务最繁重 |

**官方组织职务：**

| 职务         | 行动点上限减少 | 说明               |
| ------------ | -------------- | ------------------ |
| 校团委干事   | -2             | 行政事务较多       |
| 校团委副书记 | -4             | 核心管理层         |
| 党建班学员   | -1             | 定期学习和思想汇报 |

**叠加示例：**

| 身份组合                         | 行动点上限         | 剩余可用                   |
| -------------------------------- | ------------------ | -------------------------- |
| 无任何职务                       | 20                 | 20 点自由支配              |
| 班长                             | 20 - 3 = 17        | 日常够用，略紧             |
| 班长 + 社长                      | 20 - 3 - 2 = 15    | 明显吃紧                   |
| 班长 + 学生会主席                | 20 - 3 - 5 = 12    | 非常紧张，需取舍           |
| 班长 + 学生会主席 + 校团委副书记 | 20 - 3 - 5 - 4 = 8 | 极度紧张，几乎无暇顾及其他 |

> **设计意图：** 职务带来领导力、结算积分、高星结局条件等收益，但代价是行动点上限被压缩。玩家必须在「追求高成就职务」和「保留自由行动空间」之间做权衡。身兼数职的玩家虽然结局分数高，但日常行动捉襟见肘，更容易因无暇自习/社交/减压而滑向负面属性螺旋。

### 6.6 期末总结属性（影响结局）

每学期回合五结算时，自动汇总以下四项维度，生成该学期的**学期评分**，并显示成绩单给玩家。

| 维度                           | 计算方式                        | 说明                                         |
| ------------------------------ | ------------------------------- | -------------------------------------------- |
| **学业分（GPA）**        | 当前累计 GPA × 1000            | GPA 4.0 → 4000 分，3.0 → 3000 分，以此类推 |
| **人际分（NPC 好感度）** | 所有已认识 NPC 好感度之和       | 认识的 NPC 越多、关系越深，分越高            |
| **体育分**               | 体测等级加分 + 校园跑完成率加分 | 最高 +200 分，见下方细则                     |
| **特殊成就分**           | 每项成就至少 +50，重大成就更高  | 见下方成就列表                               |

**体育分细则：**

| 项目         | 条件                        | 加分 |
| ------------ | --------------------------- | ---- |
| 体测等级     | 优秀                        | +100 |
| 体测等级     | 良好                        | +75  |
